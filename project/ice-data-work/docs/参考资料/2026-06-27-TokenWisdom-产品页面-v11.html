工作产出物 · PRD 文档

v1
1. 产品定位

TokenWisdom 是个人工作者的半自主 AI 分身系统，长期学习用户并代表用户与外部专业执行 Agent 协作。

2. 页面层级
  • Workstation：展示定时任务、当前对话任务、最近任务和核心产物。
  • Data Analysis Workspace：展示分析视图和对话窗口。
  • General Workspace：展示文档和对话窗口。
  • Twin Manager：管理分身画像、记忆文件、Agent MD 和 Token 消耗。
3. 权限原则

Wisdom Twin 可以讨论、请求、协调和草拟执行；最终确认权属于用户，尤其是写文件、发消息、付费调用、上传上下文和固化记忆。

4. 首版 MVP

Web Workstation + Workspace + Twin Dock + Agent Hub + Knowledge & Artifacts + Approvals & Audit，Mac Local Runtime 作为 P1 接入本地执行 Agent。
