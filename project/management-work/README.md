# 小米中基层管理指南

> 来源：[飞书文档](https://mi.feishu.cn/docx/MT3RdVlUCo7zQ6xEzdscYEbNn9J)

## 文档结构总览

管理指南包括三大篇章：

- **理念篇**：管理活动的"思想总纲"，体现信念、价值观、思维和原则，为日常决策和行动提供方向指引
  - **管理纲领**：为什么做事
  - **管理思维**：怎么想事
  - **管理原则**：怎么做事
- **任务篇**：管理职责和活动的具体流程、步骤和动作集合
  - **管理任务**：我该做什么
- **锦囊篇**：管理者的"实战弹药箱"，源自实战经验提炼
  - **管理锦囊**：我要参考什么
