# 任务篇：管理的8项任务

## 1. 规划：如何做好管理规划？

**规划是为了明天：有什么事情如果"今天"做了，可以让"明天"更好，或者至少让"明天"不会更糟。**

### 操作方法：五步法

### 第一步：梳理团队职能

管理者首先要回答"团队是谁？"。使用《团队定位分析输入表》从四个维度出发收集与分析信息。

**公式**：（团队名称）将通过（贡献：团队为达成目标将运用的手段/做的事情），成为（定位：团队的角色），从而支持（价值：支撑的组织目标）。

> **示例：**（供应链管理团队）将通过（供应商分级管理、长周期物料备货策略、库存周转率优化），成为（资源整合者），从而支持（优化采购成本与供应稳定性，实现BOM成本同比下降8%-12%）。

### 第二步：找到关键目标

回答"团队要去向哪里"。关键目标是对业务价值更大、优先级更高的事项，是决定团队一年核心战功的目标。

**关键目标的原则：**
- 聚焦重点：聚焦最重要的方向，最多不要超过5个
- 富有挑战：让人产生"不舒服的兴奋感"
- 简洁描述：一句话描述清楚，便于记忆

### 第三步：拆解关键目标

推荐拆解"引领性指标"而非物理式拆解指标。引领性指标有两个核心特点：
- **预见性**：根据它的变化，可以推断出关键目标的相应变化
- **可控性**：它可以被你的团队直接控制和影响

**引领性指标的原则：**
- 可衡量：尽量定量描述，要有数字支撑
- 要聚焦：避免罗列所有日常工作，不超过5条

### 第四步：盘点团队人员

从业务分工和人员能力两个维度出发盘点"谁能重点使用，谁需要长期培养，谁不适合需要调整工作，又需要引进什么样的人"。

### 第五步：与上级沟通共识

规划需要上级把关，也需要获得上级的支持。达成共识后，把规划分享给团队以及业务相关方。

> **总结**：一线管理者要切忌"基于现有的资源做规划"，需要以终为始"基于远方的目标回推来做规划"。

## 2. 招聘：如何做好面试识人？

**招对人是管理的起点，能解决80%的管理问题。**

### 操作方法：四步法

### 第一步：梳理人才画像

推荐使用冰山模型来描绘人才画像。

**使用冰山模型的tips：**
- 冰山上：基本条件符合即可，没必要过度拔高。过度拔高有可能过滤掉高潜力的人，更高条件的人期待更高，不稳定风险更大
- 冰山下：价值观、自我形象、个性决定了适配度。动机决定了稳定性和工作投入度
  - 跳槽是动机的体现，频繁跳槽会成为一种习惯，建议不看1年1跳、2年3跳的人
  - 建议重点看近三段职业发展呈上升状态的应聘者

### 第二步：撰写招聘JD

**撰写JD三原则：**
- 候选者视角：少谈"我们"，多谈"你"；少谈"你要做什么"，谈"你会得到什么"
- 言简意赅：不宜过长，易于阅读，可以一口气读完
- 体现亮点：提炼公司、业务、岗位的亮点，展示岗位给候选人带来的好处

### 第三步：面试评估

推荐使用STAR行为面试法，行为面试法预测未来业绩的准确度为55%，而传统面试方法的准确度仅为14%。

**STAR面试法提问原则：**
- 多问开放性问题，少问封闭式与诱导性问题
- 多问体现正面行为的问题，少问理论式与假设性问题
- 多追问，获得更深入的了解

**面试过程注意事项：**
- 给候选人思考问题的时间，克制打断的冲动
- 留意候选人提出的问题：高潜力候选人往往有很强的好奇心
- 慎用压力式面试法，不要引起争吵

### 第四步：维护与吸引优秀候选人

- **聊顾虑**：站在对方的角度思考问题，多询问对方的职业"痒点、爽点、痛点"
- **谈未来**：介绍小米的优势、发展前景、业务前景和岗位重要性
- **勤沟通**：2天沟通一次近况，每个月聊一次业务，每次都输出一些小米的发展利好

> **总结**：找人不是三顾茅庐，找人要三十顾茅庐。如果找不到人，其实只有一个原因，就是没有花足够的时间。

## 3. 任务：如何合理科学的分配任务？

**团队业绩的达成不能只靠管理者自己，而要靠整个团队。**

### 操作方法：四步法

### 第一步：分配任务前做到心中有数

- **判定优先级**：从收益-损失两个维度来判定，损失越大、收益越高的事情越应该提升优先级
- **分析任务**：使用《工作任务分析表》分析任务复杂度和难度
- **分析人员**：谁有能力承接？谁能通过这个任务获得成长？
- **人事匹配三原则**：
  - 原则1：任务对接受者要尽量有一定的挑战
  - 原则2：老人做新事，新人做老事
  - 原则3：工作复杂度-员工熟练度矩阵

**工作复杂度-员工熟练度矩阵：**

| | 高员工熟练度 | 低员工熟练度 |
|---|---|---|
| **高任务复杂度** | 给建议，管控节点，下属自主开展 | 倾听意见并辅导，严格管控过程 |
| **低任务复杂度** | 全权委托，管控结果 | 告知如何完成，管控过程 |

### 第二步：分配任务（五步法）

1. **说明任务**：交代背景、意义和期望
2. **阐述期望**：明确交付标准和时间节点
3. **说明支持**：告知可获得的资源和支持
4. **明确检查**：约定检查方式和频率
5. **澄清共识**：让对方复述理解，确认对齐

### 第三步：阶段抽查

五种抽查形式：管理看板、日/周报、周例会、1V1沟通、走动式管理

### 第四步：完成任务后要及时总结

形成案例、工具、流程、SOP等内容，把经验转化成团队的能力。

> **总结**：优秀的任务分配"既要让任务能被很好的完成，又要让下属能够获得成长"。管理者一定要对团队成员多一些耐性。

## 4. 开会：如何组织团队内部会议？

**会议是从事管理工作必需的媒介，管理者绝对无法避免开会，但可以让会议更有效率。**

### 场景一：办公例会

- **会议目的**：了解目标进度，工作完成情况，问题和风险，下周计划
- **会议时间**：最好30分钟，最长不超过60分钟
- **参会人员**：直接管理的团队所有成员
- **会议频率**：通常每周一次，固定时间
- **会前准备**：团队成员撰写周报

**周报模板：**

| 上周工作重要进度 | 风险问题 | 行动计划 |
|---|---|---|
| 提炼当前重要进展 | 描述可能存在的问题/风险 | 规划下周具体任务 |

**会中引导：**
- 快速飞阅：全员静默阅读文档，通过评论提出疑问/建议
- 轮流汇报：围绕目标、结果、卡点、下一步计划展开
- 同步信息：传达上级部门重要通知、业务进展

**会后跟进：** 待办落实 + 进度追踪

### 场景二：主题共创会

- **会议目的**：对尚无明确结论的事件，通过集体智慧充分讨论，产出明确决策方案
- **会议时间**：2-4小时
- **参会人员**：5-10人，确保充分参与
- **会前准备**：定主题、定人员、定流程

**参考议程（130分钟）：**
1. 破冰活动：10分钟
2. 背景与信息输入：15分钟
3. 发散讨论：60分钟
4. 聚类收敛：20分钟
5. 优先级排序：10分钟
6. 计划行动：10分钟
7. 总结确认：5分钟

> **开会的8个凡是：**
> 1. 凡是会议，必有主题
> 2. 凡是会议，必有议程
> 3. 凡是会议，必限人员
> 4. 凡是会议，必有准备
> 5. 凡是会议，必须守时
> 6. 凡是会议，必有记录
> 7. 凡是会议，必有结果
> 8. 凡是会议，必有追踪

## 5. 绩效：如何做好绩效管理？

**通过绩效管理帮助团队成员了解自己的绩效情况，同时为他们提供一个改进和提升的机会。**

### 操作方法：四步闭环

### 第一步：绩效目标设定

- **拆解对齐**：自上而下逐级分解 + 自下而上逐级承诺
- **设定目标**：符合SMART原则
  - 能量化的量化：结果可直接用数据衡量
  - 不能量化的细化：过程性、能力性等难以直接量化的目标
  - 不能细化的关键事件化：项目性质的目标和探索性目标

### 第二步：绩效过程辅导

从工作进度视角、情绪状态视角和绩效表现视角判断辅导时机。推荐使用五步谈话法。

### 第三步：绩效结果评价

**评价原则：** 奖励功劳，鼓励勤奋，但更重要的是奖励最终结果

**评价内容：** 战功评级 + 内功评级 → 综合评级

| 战功评级 | 内功评级 | 综合评级 |
|---------|---------|---------|
| S：远超预期 | A：有较多积累和明显影响 | S：远超预期，是公认的楷模 |
| A：明显超出预期 | B：有积累和影响 | A：明显超出预期，广获好评 |
| B+：部分超出预期 | C：无积累 | B+：部分超预期 |
| B：满足预期 | | B：符合预期 |
| B-：部分不达预期 | | B-：部分不达预期 |
| C：不达预期 | | C：不达预期 |

- 内功为A时，综合等级在战功等级基础上上浮1档
- 内功为B时，综合等级等于战功等级
- 内功为C时，综合等级在战功等级基础上下浮1档

**末位淘汰原则**：原则上2次B-和一次C将进入汰换名单。

### 第四步：绩效结果沟通

**万能公式 = 破冰 + 对齐评价 + 共识目标 + 提供支持**

- 破冰：营造坦诚和安全的氛围
- 对齐评价：引导回顾，坦诚、客观地给出结果和理由
- 共识目标：明确接下来的重点目标和改进方向
- 提供支持：了解困难和诉求，尽可能提供相应资源和支持

> **总结**：绩效考核不是终点，成长才是目的。只有做到"目标-辅导-评价-沟通"的完整闭环，才能真正提升团队整体的活力。

## 6. 激励：如何做好非物质激励？

**下属是否感受到上级真正关心自己，是激励员工的一个重要因素。**

### 操作方法：三步法

### 第一步：识别下属的核心需求

使用3F沟通法：Fact（了解事实）→ Feel（了解情绪）→ Focus（了解意图与诉求）

**人的底层动机三大类别：**
- **成就需求**：争取成功，渴望将事情做得更完美 → 更需要"尊重、发展、意义"
- **权力需求**：影响或控制他人且不受他人控制 → 更需要"责任、授权、自由"
- **亲和需求**：建立友好亲密的人际关系 → 更需要"信任、安全、归属"

> Tips：每个人在不同阶段第一核心诉求会有所调整。应届生往往需要归属感，社招专家往往需要意义感，团队骨干往往需要授权。

### 第二步：盘点非物质激励的资源

三个层面的激励资源：
- **团队氛围层面**：信任、公平、成果
- **员工发展层面**：目标激励、展示机会、学习机会、工作授权、专长认可
- **工作关系层面**：以身作则、参与研讨、用人之长、懂得包容、多倾听多认可

### 第三步：匹配激励策略

推荐使用BET反馈法进行激励型认可反馈：
- **B（Behavior）行为**：描述具体的行为
- **E（Effect）效果**：说明行为带来的效果
- **T（Thanks）感谢**：表达感谢

> **总结**：管理者要有发现"美"的眼睛，不遗余力地找机会给下属鼓掌。

## 7. 总结：如何做好工作总结？

**总结是价值的放大器，不能形成团队进化的总结都是"伪总结"。**

### 操作方法：复盘 + 呈现

### 第一步：复盘

1. **评估结果**：通过《结果与目标对比表》找到亮点和不足
2. **还原过程**：重现关键行为、关键决策、团队状态
3. **分析原因**：
   - 分析成功事件：多列举客观因素
   - 分析失败事件：多从自身找原因
   - 分析根本原因：使用5Why分析法
4. **总结经验**：找到经验背后的规律和认知
5. **应用推广**：开始做、继续做、停止做

### 第二步：呈现（结构化四原则）

1. **结论先行**：每个话题只有一个结论且放在最前面
2. **以上统下**：任何一个层次上的观点都必须是其下一层次观点的概括
3. **归类分组**：每组中的观点必须属于同一个范畴（MECE原则：相互独立+完全穷尽）
4. **逻辑递进**：每组中的观点都必须按照某种逻辑顺序组织

> **总结**：复盘不是目的，目的是持续突破认知，带来持续的成长；结构化呈现也不是目的，目的是提高听众理解内容的效率。

## 8. 培训：如何做好团队的培训？

**管理者要把培训作为日常的事情看待。**

### 操作方法：三步法

### 第一步：识别培训时机

- **能力差距导致的业绩差距**：从业绩差距中找到行为表现差距，针对"不知道"和"不会"的进行培训
- **下属面临新角色/新事情**：刚入职的应届生、转岗同学、新业务/项目
- **优秀实践经验**：成功完成项目后的方法经验需要推广

### 第二步：盘点培训资源

- 通用软技能：通过清河大学在线学习
- 专业技能：一线管理者组织业务骨干共同开发，建立团队专属知识库

### 第三步：实施工作辅导

**场景一：初级下属的辅导 — 在岗辅导四步法**

1. 我说你听：主管介绍任务的操作要点和背后的原理
2. 你说我听：下属通过自己的语言陈述对任务的理解
3. 我做你看：主管示范，亲自操作，手把手带教
4. 你做我评：下属操作任务，主管进行反馈

**场景二：资深下属的辅导 — 教练辅导四步法（GROW模型）**

1. **Goal（目标）**：明确本次辅导对话希望达成的具体目标
2. **Reality（现状）**：引导对方客观描述当前情况与障碍
3. **Options（选择方案）**：探索可能的解决方案，通过提问激发对方自己想出方案
4. **Way Forward（行动计划）**：引导对方从选项中选择具体、可执行的下一步行动

> **总结**：培训的效果并非立竿见影，但其一旦显现，将为管理者带来巨大效益。"培训是管理的高杠杆活动"，一线管理者在员工培训上一定要秉持长期主义的理念。
