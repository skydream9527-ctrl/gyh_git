# 潜在结果框架（Potential Outcome Framework）

> 又称 Rubin 因果模型（RCM）。本文目标：让你能用这套语言**精确说出**一个因果问题在问什么，并理解为什么"因果效应不可观测"是个根本困难、人类怎么绕过它。

---

## 一、为什么从"潜在结果"开始

因果推断最大的诅咒是**词不达意**：

- "广告涨了 5%" — 涨了什么？相比谁？是同一群人吗？
- "用了 X 功能的留存高" — 用了 X 是 X 的功劳，还是用 X 的人本来就更活跃？

潜在结果框架（Donald Rubin, 1974）提供的就是一套**消歧的语言**——它把因果问题表达得清晰到能写数学。这套语言现在是因果推断界的**通用语**，所有现代因果方法都建立在它之上。

---

## 二、核心定义：每个个体两个"潜在结果"

对每个个体 i，定义：

```
Y_i(1) = 个体 i "如果接受处理（T=1）" 的结果
Y_i(0) = 个体 i "如果未接受处理（T=0）" 的结果
```

记号说明：
- **T**（Treatment）= 处理变量，例：是否投广告、是否上线新功能
- **Y**（Outcome）= 关心的结果，例：是否购买、留存天数、GMV
- **Y(1)、Y(0)** 都叫**潜在结果（potential outcomes）**——每个人在两种情况下"潜在地"会发生的事

### 个体因果效应（ITE）

```
τ_i  =  Y_i(1) − Y_i(0)
```

这就是我们真正想要的：**对个体 i 来说，处理产生了多少差异。**

### 因果推断的根本问题（Fundamental Problem of Causal Inference）

你**永远只能观测到 Y_i(1) 和 Y_i(0) 的其中一个**，因为同一个人在同一时刻不可能既被处理又没被处理。

```
观测到的结果：
  Y_i = T_i · Y_i(1) + (1 − T_i) · Y_i(0)

T_i = 1  →  我们看到 Y_i(1)，看不到 Y_i(0)
T_i = 0  →  我们看到 Y_i(0)，看不到 Y_i(1)

→ ITE τ_i 永远观测不到（除非时光机或克隆人）
```

这是因果推断的命门。后续所有方法都在解决一件事：**用别人的 Y(0) 当成你的反事实**。

---

## 三、四种因果效应：估计目标的"度量衡"

ITE 不可观测，那就退而求其次估群体平均。**不同群体的平均给出不同的因果量**：

| 估计量 | 全名 | 定义 | 业务含义 |
|---|---|---|---|
| **ATE** | Average Treatment Effect | E[Y(1) − Y(0)] | 全体用户的平均效应（"如果给所有人都上"） |
| **ATT** | Avg. Treatment Effect on the Treated | E[Y(1) − Y(0) \| T=1] | 已处理群体的平均效应（"对真投了的那批人，广告值不值"） |
| **ATC** | Avg. Treatment Effect on the Control | E[Y(1) − Y(0) \| T=0] | 未处理群体的平均效应（"如果对没投的那批人投了，能涨多少"） |
| **CATE** | Conditional Avg. Treatment Effect | E[Y(1) − Y(0) \| X=x] | 给定特征 x 的子群效应（个性化干预的基础） |

### 一个可能让你警觉的事实

**ATE、ATT、ATC 通常 ≠**。例如广告投放：
- 平台只对"高购买倾向"用户投广告 → ATT 看起来很高（这些人本来就要买）
- 但 ATE（如果对所有人都投）会低得多——因为大量低意向用户不会因为广告就买

> **报告时一定要写清你估的是哪一个**。"广告 ROI 200%" 是 ATT 还是 ATE？可能差 5-10 倍。

### CATE：个性化的基础

CATE = E[Y(1) − Y(0) | X = x]

它说的是"给定特征 x 的子群，因果效应是多少"。它是 **Uplift Modeling** 的核心目标：

- 谁是"广告的受益群"？（CATE > 0）
- 谁是"广告的反感群"？（CATE < 0，给了广告反而流失）
- 谁是"无所谓群"？（CATE ≈ 0，给不给都一样）

业务最大价值不在 ATE，而在能识别这三类人——把预算只花在第一类上。

## 四、关键假设：什么时候"群体均值"能近似"因果"

要从观测数据估上面这些量，需要一组核心假设。**这是因果推断的"小字"——任何方法都建立在这里**。

### 1. SUTVA（Stable Unit Treatment Value Assumption）

中文叫"稳定单位处理值假设"。两个子假设：

- **No Interference（无溢出）**：个体 i 的潜在结果不依赖于别人是否被处理。
- **Consistency（一致性）**：处理 T=1 在所有人身上是"同一种处理"，没有版本差异。

⚠️ 何时违反：

- **社交网络 / 双边市场**：你给 A 投了广告，A 把信息分享给 B → B 的 Y(0) 也变了。
  - 这是 Uber、Doordash、Meta 头疼的根源 → 用 Switchback、地理隔离实验。
- **多版本处理**："广告"包含 banner、信息流、视频三种 → 一致性破。

### 2. Ignorability / Unconfoundedness（可忽略性）

```
{Y(0), Y(1)} ⊥ T | X
```

人话：**给定可观测协变量 X 后，处理分配 T 与潜在结果独立**。即"X 一样的人里面，T=1 和 T=0 的差异完全是处理造成的"。

⚠️ 这是观察性研究的**最强假设**，永远无法直接验证。能做的：
- 列出尽可能多的混淆变量
- 做敏感性分析（Rosenbaum bounds）测假设破坏的容忍度
- 找"自然实验"绕开（IV、RDD、DID）

### 3. Positivity / Overlap（正性 / 重叠）

```
0 < P(T=1 | X = x) < 1, ∀ x
```

人话：**对每个特征组合，都要有处理组和对照组样本**。

⚠️ 何时违反：
- 老人组里全是没投广告的，年轻组里全是投了的 → 老人组的 Y(1) 没有可比对象 → 估不出。
- 解法：剪掉重叠区外的样本（牺牲外推性），或重新定义研究人群。

### 4. Consistency / SUTVA 中的"处理良定义"

T=1 必须有清晰、可观测、可执行的定义。"减肥" 这种模糊处理是因果推断的天敌。AB 测试里的 T 是按钮上没上线，相对干净。

## 五、随机化为什么是黄金标准

如果 T 是**随机分配**（AB 测试），上面的假设几乎自动满足：

- 随机分配 → T 与任何变量独立 → 无混淆 → Ignorability 自动成立
- 50/50 分流 → 任何特征组合的两组都有样本 → Positivity 成立
- 两组在 X 上**期望相同** → 直接 E[Y|T=1] − E[Y|T=0] 就是无偏 ATE

```
ATE 在 AB 测试中的简单估计：
   ̂ATE = mean(Y | T=1) − mean(Y | T=0)
```

> 这就是为什么 AB 测试是因果推断的金标准——它从设计上"造出来" Ignorability。

观察数据没这个奢侈，所以才有了 PSM、IPW、DML 等一堆方法去**模拟随机化的效果**。

## 六、把潜在结果用在实际场景

### 场景 1：评估广告增量

```
T = 1 (投广告) / 0 (没投)
Y = 是否购买（0/1）

ATE = "如果对所有人都投广告，购买率会涨多少？"
ATT = "对实际投了广告的人，广告带来了多少购买？"
CATE(x) = "对特征 x 的子群（如年轻女性），广告带来多少？"
```

错误做法：直接对比"看了广告的人"和"没看的人"购买率——平台投放本身就有偏（投给了高意向人群）。

正确思路：找混淆变量 X（历史购买、活跃度…）→ 检查 Positivity → 用 PSM 或 DML 估 ATT/ATE。

### 场景 2：评估补贴券

```
T = 是否发券
Y = 30 天 GMV

业务真正想知道的是：CATE
  - 哪些用户给券会带来增量？（敏感型）
  - 哪些用户不给也会买？（自然成单型，给券=亏钱）
  - 哪些用户给了也不买？（无效型）
```

这就是 Uplift Modeling 的入口——直接对 CATE 建模、按 CATE 排序选人。

### 场景 3：LLM 改了一个 prompt，效果变好了

T = 用新 prompt / 用旧 prompt
Y = 用户满意度评分

⚠️ 看起来能 AB 测——可以！但小心：
- SUTVA：用户不会互相影响吗？（多人聊天产品就会）
- Consistency：T=1 真的统一吗？（prompt 内随机性、温度参数）

很多 LLM 团队跳过这步直接做 vibe-check，本质是放弃因果识别。

## 七、把潜在结果框架装进脑子的练习

下次看到任何"X 涨了 Y" 的报告，强迫自己回答：

```
1. T 是什么？怎么定义的？
2. Y 是什么？观测到的是 Y_i(T_i)，对吗？
3. 报告的是 ATE / ATT / ATC / CATE 中的哪个？
4. 反事实是用谁的 Y(0) 当替代的？
5. SUTVA 成立吗？Ignorability 成立吗？Positivity 成立吗？
6. 哪个假设最可能破？最坏情况下结论会反吗？
```

回答完这 6 个问题之后再决定是否信这个报告。这是从"看数据"升级到"看因果"的关键转变。
