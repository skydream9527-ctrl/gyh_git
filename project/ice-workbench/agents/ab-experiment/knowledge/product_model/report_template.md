# AB实验分析报告模板

标题格式：`AB实验分析报告 - {experiment_id} - {phase}阶段`

## 报告头信息

- 实验ID
- 关键变量
- 放量阶段
- 分析日期
- 数据时间范围

***

## 第1节：结论（必须）

结论先行，明确给出放量/不放量/继续观察的判断

### 内容要素

- 放量建议
- 核心理由
- 关键数据支撑，全部核心指标+关键直接指标（1～3个），表格呈现，含实验组及对照组的绝对值及相对比变化

### 角色适配

| 角色  | 展示内容           | 详细程度 |
| --- | -------------- | ---- |
| PM  | 结论 + 建议 + 关键数据 | 摘要   |
| 分析师 | 结论 + 建议 + 统计依据 | 详细   |

***

## 第2节：核心数据表现（必须）

核心指标全量数据，标注变化方向与幅度。

### 内容要素

- 规模指标：方向 + 幅度 + 置信区间
- 消费指标：方向 + 幅度 + 置信区间
- 收入指标：方向 + 幅度 + 置信区间
- 可视化图表

### 图表要求

- 柱状图：各核心指标变化幅度对比图
- 数据表：核心指标详细数据表（对照组均值、实验组均值、变化幅度、p值、置信区间）

### 角色适配

| 角色  | 展示内容                     | 详细程度 | 特殊规则            |
| --- | ------------------------ | ---- | --------------- |
| PM  | 方向 + 幅度 + 图表             | 摘要   | 隐藏：p值、置信区间、检验方法 |
| 分析师 | 方向 + 幅度 + 置信区间 + p值 + 图表 | 详细   | 展示全部            |

***

## 第3节：直接指标数据表现（必须）

直接指标数据，须指向并能解释核心指标变化。

### 内容要素

- 各直接指标：方向 + 幅度 + 置信区间
- 因果链路验证结果
- 可视化图表

### 图表要求

- 柱状图：各直接指标变化幅度对比图
- 流程图：因果链路验证流程图（关键变量→直接指标→核心指标）

### 角色适配

| 角色  | 展示内容              | 详细程度 |
| --- | ----------------- | ---- |
| PM  | 方向 + 幅度 + 是否符合预期  | 摘要   |
| 分析师 | 完整统计细节 + 因果链路验证过程 | 详细   |

***

## 第4节：下钻分析（条件触发）

**触发条件：** 核心指标下降 OR 直接指标不符合预期

当核心指标或直接指标不符合预期时，分析原因。

### 下钻规则

- **必须列出≥3个竞争假设**，每个假设需有明确的数据验证方案
- 假设按可信度排序（⭐评级），标注数据支撑强度
- 禁止仅给出单一假设作为结论
- "分流偏差"假设不得优先于有物理/业务逻辑支撑的假设
- 量化归因：每个假设需尽量给出"该假设可解释总变化的X%"

### 内容要素

- 竞争假设列表（≥3个，含可信度评级）
- 各假设的数据验证结果
- 量化归因分析（解释比例）
- 问题定位与综合结论
- 可视化图表

### 图表要求

- 柱状图：下钻维度数据对比图
- 数据表：下钻维度详细数据表

### 角色适配

| 角色  | 展示内容                | 详细程度 |
| --- | ------------------- | ---- |
| PM  | 问题定位 + 业务建议         | 摘要   |
| 分析师 | 假设 + 数据 + 定位 + 统计细节 | 详细   |

***

## 第5节：附录（必须）

补充信息，供需要深入了解的用户查阅。

### 内容要素

- AA基线扣除详情
- SRM校验结果
- 统计功效校验结果
- 统计检验方法说明
- 数据来源说明

### 角色适配

| 角色  | 展示方式 | 详细程度 | 默认状态 |
| --- | ---- | ---- | ---- |
| PM  | 折叠   | 最简   | 默认收起 |
| 分析师 | 展开   | 详细   | 默认展开 |

***

## 报告尾信息

- 分析工具版本
- 知识库版本
