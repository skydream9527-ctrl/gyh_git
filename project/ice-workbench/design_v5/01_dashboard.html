📐 这张稿严格沿用真实代码 [DashboardPage.tsx](../frontend/src/pages/dashboard/DashboardPage.tsx) 的结构

  • 顶栏：[TopNav](frontend/src/components/shell/TopNav.tsx) mode="dashboard"，brand + nav + rightActions（4 按钮：新任务 / 定时 / 指南 / 管理）
  • 1. dash-welcome：👋 你好，{user?.name} + 选一个范式开始今天的工作 —— 字号 36px Fraunces 衬线
  • 2. dash-start (.start-grid)：4 列等宽，每张 .start-card padding 18×20，左侧 3px 彩色条边（起点不同色：accent / data / biz / ab）
  • 3. ⚡ 快速开始 (.paradigm-grid)：auto-fit minmax(180px, 1fr)，paradigm-card 固定 height 160，icon 44×44 radius 11，border-top: 3px solid 范式色。已上线在前 + 待上线分组（opacity 0.55）
  • 4. 📋 我的任务 (.task-grid)：5→4→3→2 响应式，task-card min-h 132，padding 18，标题 Fraunces 15/500，tc-paradigm 用 mono uppercase
  • 5. 🌐 团队公共区 (.public-tabs + .public-grid)：5 Tab（pill 形）+ 6→4→3→2 列响应式。激活 tab 用 accent 实色 + 反白
  • FAB：浮动右下，仅移动端显示，accent 实色
  • 底栏 .p-tabbar：4 Tab（首页 / 任务 / Agent / 我的），仅移动端

和 v4 关键差异

  • 底色：纸张白 #faf9f6 → 暖米白 #faf4ed（更暖）
  • Accent：克制蓝 #1a73e8 → Claude 铜橙 #cc785c（实色，无渐变）
  • 字体：Inter 全场 → Fraunces 衬线（标题 / 任务名 / 范式名 / 起点卡名）+ Inter 正文 + JetBrains Mono chip
  • 形状：胶囊 99px → 方角 4-11px（卡 10/11、按钮 8、chip 4）
  • chip 文字：中文 → Mono 大写英文（AB / BIZ / DATA）
  • 按钮形状：胶囊 → 8px 方角（顶栏的「+ 新任务」「⏱ 定时」按钮也变方）
  • 布局尺寸：完全沿用 [Dashboard.css](frontend/src/pages/dashboard/Dashboard.css) 真实尺寸（dash-main 1200max / paradigm-card 160h / task-card 132min-h / start-grid 4 列 / public-grid 6→2 列）
