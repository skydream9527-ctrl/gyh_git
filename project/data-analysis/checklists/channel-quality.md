# 渠道质量评估 Checklist

> 评估获客渠道质量，决定加量、减量还是停投。

## 量级与成本

- [ ] **量级多大？** 日均获客数、获客趋势
- [ ] **成本多少？** CAC、成本趋势、和行业基准对比

## 留存与质量

- [ ] **次留 / 7 留 / 30 留怎么样？** 和自然量对比差多少
- [ ] **留存后行为**：渠道用户留存后的活跃度、消费深度，和自然量差异
- [ ] **LTV 多少？** 多久能回本（LTV / CAC ≥ 3 为健康）
- [ ] **有没有作弊 / 刷量？** 设备指纹、IP 集中度、行为异常（秒级操作、规律性点击）

## 决策

- [ ] **加量还是减量还是停投？**
- [ ] **优化方向**：素材、定向、出价、落地页
- [ ] **预算分配**：高质渠道加预算，低质渠道降预算

## 评估维度表

| 维度 | 指标 | 判断 |
|------|------|------|
| 量级 | 日均获客 | 是否达标 |
| 成本 | CAC | 是否低于 LTV/3 |
| 短期留存 | D1 / D7 | 是否接近自然量 |
| 长期留存 | D30 | 是否达标 |
| 价值 | LTV | 回本周期 ≤ 6 月 |
| 真实性 | 作弊率 | < 5% |
