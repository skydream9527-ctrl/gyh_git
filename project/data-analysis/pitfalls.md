# 数据分析常见陷阱与避坑指南

> 8 个高频陷阱，每个都曾导致过错误决策。分析时逐项排查。

## 陷阱速查表

| 陷阱 | 表现 | 后果 | 避坑方法 |
|------|------|------|----------|
| **相关当因果** | A 和 B 一起涨就认为 A 导致 B | 决策方向完全错误 | 找混淆变量、控制变量、AB 测试验证 |
| **幸存者偏差** | 只看成功 / 留存用户的特征 | 结论完全相反 | 必须对比全量用户，分析流失样本 |
| **辛普森悖论** | 总体和各分组结论相反 | 看到假趋势 | 必须分层下钻，不要只看总数据 |
| **口径不一致** | 同一个指标不同人算出来不一样 | 鸡同鸭讲，结论矛盾 | 建立指标字典，口径文档化，对比前先对齐 |
| **忽略基数 / 样本量** | 只看转化率不看流量，小样本下结论 | 高估小群体重要性，把波动当结论 | 相对数 + 绝对数一起看，做统计显著性检验 |
| **过度拆解** | 拆到太细的维度找不到重点 | 分析半天看不出结论 | 先算贡献度，优先拆贡献最大的维度 |
| **为解释而解释** | 强行给正常波动找原因 | 自己吓自己，做无用功 | 先判断是否在正常波动区间（±3σ），不是所有涨跌都有原因 |
| **确认偏差** | 只看支持自己观点的数据，忽略反面 | 选择性失明，结论偏见 | 主动找反例，让不同观点的人挑战结论 |

## 重点陷阱详解

### 相关 ≠ 因果
两个指标同步变化不代表因果关系。夏天冰淇淋销量和溺水人数都涨，不是冰淇淋导致溺水，是夏天这个混淆变量。

**验证因果的方法**：
- 控制变量：固定其他变量看关系是否成立
- 时序验证：原因是否在结果之前
- AB 测试：随机分组干预，金标准

### 辛普森悖论
分组结论和总体结论相反。例：新版整体转化率高于旧版，但分渠道看每个渠道都是旧版高——因为新版用户里高转化渠道占比更多。

**规避**：分析转化率、留存率等比率指标时，必须同时看分群和总体，并关注各群样本量。

### 幸存者偏差
只分析留存用户特征，得出"使用 X 功能的用户留存高"——可能是留存下来的用户才用 X，不是 X 导致留存。

**规避**：对比流失用户和留存用户的行为差异，用首次行为预测后续留存。
