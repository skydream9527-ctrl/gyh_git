# 归因分析（Attribution Analysis）

## 什么是归因分析
将转化/价值按照一定规则分配给用户路径上的各个触点/渠道，评估每个触点/渠道的贡献，用于渠道投放优化、营销资源分配。

## 常见归因模型
1. **末次触点归因**：功劳100%给最后一个触点 → 简单，但忽略前面的所有种草/教育环节
2. **首次触点归因**：功劳100%给第一个触点 → 适合拉新评估，忽略转化临门一脚
3. **线性归因**：功劳平均分给路径上所有触点 → 公平但没有重点
4. **时间衰减归因**：越靠近转化的触点功劳越大 → 适合长决策周期产品
5. **U型归因**：首次触点40%，末次触点40%，中间触点分20% → 平衡拉新和转化
6. **数据驱动归因**：用算法/模型（如Shapley值、马尔可夫链）根据实际数据计算每个触点的真实贡献 → 最准确，但需要足够数据量
