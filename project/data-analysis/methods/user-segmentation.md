# 用户分群（User Segmentation）

## 什么是用户分群
根据用户的属性、行为、价值等特征，把用户分成不同的群体，对不同群体用差异化的产品/运营策略，比"千人一面"效率高很多。

## 常用分群维度
1. **属性分群**：年龄、性别、地域、设备、职业、注册时间
2. **行为分群**：活跃度、使用时长、功能使用偏好、行为路径
3. **价值分群**：RFM分层、LTV、付费金额、贡献度
4. **生命周期分群**：新手、成长、成熟、衰退、流失
5. **需求分群**：用户来产品的核心目的/需求类型

## 分群方法
- **规则分群**：按明确规则划分（如RFM、生命周期），简单易解释，业务最常用
- **算法分群**：聚类算法（K-Means/DBSCAN）、预测模型，适合复杂场景
