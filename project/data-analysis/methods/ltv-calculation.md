# LTV/CLV 用户终身价值计算

## 什么是LTV
Life Time Value，用户终身价值，指用户从注册到流失整个生命周期内为产品创造的总价值（收入/利润）。
核心用途：判断获客成本（CAC）是否合理，LTV/CAC ≥ 3是健康的商业模式基准。

## 核心公式
简化公式：`LTV = ARPU × 用户生命周期（月/天）`
更精准：`LTV = Σ（第n个月留存率 × 第n个月ARPU）`，按生命周期求和

## 关键指标
- **CAC（用户获客成本）**：获取一个新用户的平均成本 = 总营销费用/新增用户数
- **ARPU**：每用户平均收入 = 总收入/活跃用户数
- **ARPPU**：每付费用户平均收入 = 总收入/付费用户数
- **回本周期**：多长时间LTV能覆盖CAC，越短越好
