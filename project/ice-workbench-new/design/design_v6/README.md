# ICE Workbench v6 设计规划讨论与设计稿

本目录用于探讨与规划 ICE Workbench 的 v6 版本功能演进与对应的页面视觉/交互设计。

## 讨论方向与目标

1. **功能演进**：讨论需要新增的重磅功能或需要重构、重划定的核心模块。
2. **页面设计与风格 (Design)**：在目前 v5 (Claude Code / Anthropic 暖底风格) 的基础上进行体验升级、交互梳理，抑或向其他设计语言（如 Moxt、Notion、Linear 等）全面转化。
3. **架构与性能**：任何涉及前端架构与后端通信模式变化的设计决策。

---

*随着讨论深入，我们将在本目录下逐渐丰富各模块的具体设计草图 (`.html` Demo 或 `.md` 文档)。*
