快速上手

ICE Workbench 是一个基于 Agent 范式的智能数据工作台。本指南帮你在 5 分钟内完成第一个数据分析任务。

三步开始

1

选择 Agent

在首页选一个专业 Agent，或从范式新建。

2

描述需求

用自然语言说清你想分析什么。

3

查看结论

Agent 自动取数、计算并产出图表结论。

核心概念

理解这几个概念，能让你更高效地使用工作台：

  • 范式（Paradigm）：决定 Agent 的默认工具集与输出模版，如 data / ab / biz。
  • Agent：在某个范式下、带专业 prompt 与知识库的智能体。
  • Skill：可复用的子流程，挂载到 Agent 上扩展能力。
  • 任务（Task）：一次完整的分析会话，包含对话、文件与产物。

小技巧：在对话框输入 @ 可以快速引用已上传的文件，输入 / 唤起指令菜单。

创建第一个任务

点击右上角「+ 新任务」，选择「数据分析」范式与「数据分析师」Agent，输入你的需求，例如「帮我统计上周各渠道的次日留存」。进入 Workspace 后，Agent 会先确认数据口径，再自动生成 SQL 并执行。整个过程在右侧执行台实时可见。
