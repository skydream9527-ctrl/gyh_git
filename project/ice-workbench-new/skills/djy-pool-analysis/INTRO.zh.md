# 都江堰内容池日报分析 Skill

用于都江堰内容池日报、字段校验、消费分析和创作者审核链路校验。包含日报流水线脚本、SQL 模板、飞书推送卡片生成、HTML/长图报告生成、豁免规则和参考文档。

## 适用场景

- 读取用户上传的 SQL，自动执行查询、汇总数据、生成 AI 日报，并在确认后推送飞书。
- 生成或补跑都江堰内容池日报，并按规则推送到飞书群。
- 巡检 4 家 CP 内容池字段准确性，输出异常规则和脏数据明细。
- 分析 DAU 级消费指标，如曝光、VV、时长、CTR、实验组对比。
- 校验创作者签约名单到审核表、内容池、签约清单的闭环一致性。

## 使用提醒

- 日报推送必须明确获得用户的“推送/发到飞书/发群”指令。
- 每次分析必须真实跑 SQL，不使用缓存或历史截图冒充结果。
- 运行脚本依赖 DataWorks/Kyuubi、飞书凭证、Playwright/Chrome 等本地环境；缺少配置时应明确报错并给出补齐项。
