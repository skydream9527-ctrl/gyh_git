Core Functions

一个工作台，串起从提出问题到产出报告。

核心功能围绕“创建任务、执行分析、管理上下文、沉淀资产、自动复跑”展开。 用户不需要理解底层工具链，只要在合适的范式里把问题交给 Agent。

入口与任务创建

Dashboard 提供快速开始、我的任务与团队资产；创建任务采用 3 Step 工坊，降低从想法到 Workspace 的启动成本。

01

Dashboard 快捷入口

首页聚合新建任务、定时任务、使用指南、公共任务，以及可一键创建的 Agent 范式卡片。

  • 快速开始卡片面向高频场景
  • 我的任务与协作任务集中管理
  • 团队资产以 Tab 组织，便于复用
02

3 Step 创建工坊

从空白、模板、公共任务或个人模板开始，配置范式、Agent、Skills、Prompt、可见性与可选定时。

  • 先选起点，再做能力配置
  • 支持私有或开放给团队
  • 成熟模板可一键使用
03

六种工作范式

围绕经营分析、AB 实验、数据分析、波动分析、版本灰度、开放任务建立业务语义入口。

  • 范式决定默认 Agent 与工具组合
  • 开放任务支持自由探索
  • 灰度、AB 等场景沉淀 SOP

Workspace 工作空间

Workspace 是日常使用主界面，采用三栏布局，将上下文、Agent 对话和结果预览放在同一张工作台上。

Task Workspace

对话不是孤立聊天，而是任务执行现场

左侧管理文件，中间运行对话，右侧展示 Agent、文件、会话、待办和配置。任务内的所有动作都围绕同一个 task_id 记录和复现。

文件上下文本地上传、公共文件引用、Agent 产物、飞书链接导入都可进入任务。
会话分支同一任务可维护多个 Conversation，便于尝试不同分析路径。
执行待办Agent 自动维护 TODO，用于长任务拆解、进度跟踪和结果回看。
Files文件预览支持 md/txt/csv/json/sql/py/pdf/png 等常见格式。
Chat流式对话边生成边展示，可中止长回复和工具链。
Plan只读规划Plan Mode 先输出方案，经用户确认后再执行。
Export导出对话将对话、工具摘要和结论导出为 Markdown。
Share开放团队任务经审核后进入公共任务，其他同学可 Fork。

Agent 与 Skill 能力

Agent 负责领域方法论和任务编排，Skill 负责可调用能力。两者组合出不同数据工作范式。

🤖

通用 Agent

作为入口编排者，适合不确定找谁的场景；支持派单、并行、后台任务和跨范式工具组合。

📊

数据分析 Agent

自然语言查数、Kyuubi SQL、Python 沙箱、趋势分析、预测和图表报告一体化。

🧪

AB / 火山实验

围绕实验 ID、显著性、AA 期、涨跌报告和放量建议提供结构化分析。

🌐

灰度与监控

对比版本核心指标、护栏指标和异常告警，支持灰度放量或回滚决策。

SQL / 数据

Kyuubi + NL→SQL

用自然语言生成查询、执行 SQL、获取表结构和指标口径，服务数据分析主链路。

文档 / 知识

飞书与知识库

飞书文档、知识库、图片上传、报告发布与知识检索构成团队协作出口。

计算 / 产物

Python 沙箱与文件

在隔离环境中执行分析代码，生成 CSV、图表、Markdown 报告等任务产物。

团队资产与运营后台

系统不仅服务单次分析，还把好的任务、模板、文件和经验变成可治理的团队资产。

公共任务

优秀 Workspace 可开放给团队，经 admin 审核后供其他用户 Fork 复刻。

任务模板

预配置范式、Agent、Skills、Prompt 和定时策略，降低重复任务创建成本。

公共文件

团队规范、指标口径、方法论、AB 文档可置顶，并通过 @ 文件注入上下文。

经验卡片

对话中的优秀结论可沉淀为经验，经审核后注入 Agent 知识库。

定时任务

支持 cron 配置、历史回放、日报巡检、异常告警和飞书推送。

Admin 后台

维护用户、Agent、Skills、模板、公共文件、用量成本、审计日志和系统开关。
