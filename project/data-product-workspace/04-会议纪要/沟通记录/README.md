## 记录规范

每个人物档案包含以下模块：

1. **基本信息**：角色、负责领域、协作频率
2. **沟通风格**：偏好的沟通方式、决策节奏
3. **关注重点**：核心KPI、经常关注的问题、决策倾向
4. **交互历史**：按日期记录重要沟通、决策、待跟进事项
5. **决策记录**：重要共识、决策结论、落地状态
6. **待跟进事项**：需要向对方确认/同步/汇报的事项

## 更新规则

- 每次重要沟通后及时更新对应人员档案
- 提炼可复用的协作经验和沟通模式
- 重要决策必须记录，避免后续重复确认
- 待跟进事项动态更新，完成后标记已完成
