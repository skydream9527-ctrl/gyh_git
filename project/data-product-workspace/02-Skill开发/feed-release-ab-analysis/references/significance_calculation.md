# 置信度计算公式与过程参考

## 一、检验方法选择规则

| 指标类型 | metric_type | 检验方法 | 所需输入数据 |
|---------|-------------|---------|-------------|
| 均值类 | mean | Welch's t 检验 | n₀, n₁, x̄₀, x̄₁, s₀, s₁ |
| 比率类 | ratio | 双样本比率 Z 检验 | n₀, n₁, x₀, x₁ |

---

## 二、Welch's t 检验（均值类指标）

### 2.1 适用条件

- 指标类型为 mean（均值类）
- 样本独立且近似正态分布
- 不假设两组方差相等（Welch's 修正）

### 2.2 计算公式

```
输入: n₀(对照组样本量), n₁(实验组样本量), x̄₀(对照组均值), x̄₁(实验组均值), s₀(对照组标准差), s₁(实验组标准差)

步骤1: 计算标准误
  SE = √(s₀²/n₀ + s₁²/n₁)

步骤2: 计算差异
  Δ = x̄₁ - x̄₀

步骤3: 计算 t 统计量
  t = Δ / SE

步骤4: 计算自由度（Welch-Satterthwaite 近似）
  df = (s₀²/n₀ + s₁²/n₁)² / [(s₀²/n₀)²/(n₀-1) + (s₁²/n₁)²/(n₁-1)]

步骤5: 计算 p 值（双侧检验）
  p = 2 × P(T_df > |t|)

步骤6: 计算 95% 置信区间
  CI = [Δ - t_{α/2,df} × SE, Δ + t_{α/2,df} × SE]

步骤7: 计算效应量
  s_pooled = √[(n₀-1)s₀² + (n₁-1)s₁²] / (n₀+n₁-2)
  Cohen's d = Δ / s_pooled
```

### 2.3 效应量解读

| Cohen's d | 效应大小 | 含义 |
|-----------|---------|------|
| < 0.2 | 极小 | 差异可忽略 |
| 0.2 ~ 0.5 | 小 | 有微小差异 |
| 0.5 ~ 0.8 | 中 | 有中等差异 |
| > 0.8 | 大 | 有显著差异 |

---

## 三、双样本比率 Z 检验（比率类指标）

### 3.1 适用条件

- 指标类型为 ratio（比率类）
- 样本独立
- 每组样本量足够大（n×p̂ ≥ 5 且 n×(1-p̂) ≥ 5）

### 3.2 计算公式

```
输入: n₀(对照组总数量), n₁(实验组总数量), x₀(对照组成功数量), x₁(实验组成功数量)

步骤1: 计算各组比率
  p̂₀ = x₀ / n₀
  p̂₁ = x₁ / n₁

步骤2: 计算合并比率
  p̂ = (x₀ + x₁) / (n₀ + n₁)

步骤3: 计算合并标准误
  SE_pooled = √[p̂(1-p̂)(1/n₀ + 1/n₁)]

步骤4: 计算 Z 统计量
  Z = (p̂₁ - p̂₀) / SE_pooled

步骤5: 计算 p 值（双侧检验）
  p = 2 × Φ(-|Z|)

步骤6: 计算非合并标准误（用于置信区间）
  SE_unpooled = √[p̂₁(1-p̂₁)/n₁ + p̂₀(1-p̂₀)/n₀]

步骤7: 计算 95% 置信区间
  CI = [(p̂₁-p̂₀) - Z_{α/2} × SE_unpooled, (p̂₁-p̂₀) + Z_{α/2} × SE_unpooled]

步骤8: 计算 Odds Ratio
  OR = [p̂₁/(1-p̂₁)] / [p̂₀/(1-p̂₀)]
```

### 3.3 效应量解读

| Odds Ratio | 效应大小 | 含义 |
|------------|---------|------|
| 1.0 ~ 1.5 | 小 | 差异较小 |
| 1.5 ~ 2.5 | 中 | 有中等差异 |
| > 2.5 | 大 | 有显著差异 |

## 四、显著性数据结构说明

### 4.1 均值类指标的数据字段

| 字段格式 | 说明 | 示例 |
|---------|------|------|
| `{metric}_n` | 样本量 | `avg_dur_n = 12345` |
| `{metric}_mean` | 均值 | `avg_dur_mean = 35.67` |
| `{metric}_std` | 标准差 | `avg_dur_std = 12.34` |

### 4.2 比率类指标的数据字段

| 字段格式 | 说明 | 示例 |
|---------|------|------|
| `{metric}_numerator` | 分子（成功数量） | `dau_rate_numerator = 89012` |
| `{metric}_denominator` | 分母（总数量） | `dau_rate_denominator = 123456` |
| `{metric}_value` | 比率值 | `dau_rate_value = 0.7210` |

---

## 五、统计决策规则

| 场景 | p 值 | 置信区间 | 效应量 | 结论 | 标签 |
|------|------|---------|--------|------|------|
| 强显著正向 | p < 0.01 | CI 下限 > 0 | 中/大 | ✅ 强烈建议采纳 | ✅ 显著正向 |
| 显著正向 | 0.01 ≤ p < 0.05 | CI 下限 > 0 | 小/中 | ✅ 建议采纳 | ✅ 显著正向 |
| 边缘显著 | 0.05 ≤ p < 0.10 | CI 含 0 | 小 | ⚠️ 需更多数据 | ⚠️ 边缘显著 |
| 不显著 | p ≥ 0.10 | CI 含 0 | 极小 | ➡️ 无法拒绝 H₀ | ➡️ 不显著 |
| 显著负向 | p < 0.05 | CI 上限 < 0 | 中/大 | ❌ 建议终止 | ❌ 显著负向 |

---

## 六、数字格式化规范

| 数据类型 | 格式要求 | 示例 |
|---------|---------|------|
| 比率类指标值 | 百分比，保留2位小数 | 0.2834 → 28.34% |
| 均值类指标值 | 保留2位小数 | 35.6789 → 35.68 |
| p 值 | p < 0.0001 时显示为 p < 0.0001，否则保留4位小数 | p = 0.0234 |
| 相对提升 | 正数 +X.XX%，负数 -X.XX% | +2.35%, -1.23% |
| 效应量 | 保留3位小数 | Cohen's d = 0.456 |
| 置信区间 | [下限, 上限]，比率类转百分比 | [0.19%, 2.91%] |
| t 统计量 | 保留4位小数 | t = 2.3456 |
| Z 统计量 | 保留4位小数 | Z = 1.9876 |
