# 自然语言转映射表 SQL

把用户的自然语言需求转成内容生态数仓 dim/dm/ads 映射表 SQL；覆盖浏览器主端 (BM)、浏览器信息流 (BF)、内容中心 (CC) 三条业务线，并提供搜索 (SR)、小说 (NV) 的底表/宽表指标查询。

**触发场景**：用户提到「映射表」「中间表」「维度枚举」「dim 表 / dm 表 / ads 表」「关联查询」；明确要求「执行 nl-mapping-table-sql」或「使用 nl-mapping-table-sql」。
**主要功能**：维度枚举值查询、指标聚合查询、多表关联查询；按业务线匹配主表与过滤字段；自带口径自检与异常拦截。
**注意**：BM / BF / CC 支持中间表（dm/ads/dim）+ 底表/宽表（dwm/dwd，raw- 前缀）双层数据；SR / NV 仅支持底表/宽表（raw- 前缀），无中间表与维度枚举/多表关联流程，遇到相关请求需说明范围限制并引导联系 gongyunhe。
