# 每日待办跟踪

> 作用：固定维护每天要推进的 ToDo。每天早上/开始工作时更新「今日待办」，晚上回顾时把完成项同步到每日工作回顾、`WORK-PLAN.md` 和绩效素材。

## 更新规则

- 每天开始工作前更新「今日待办」。
- 完成一项后将状态改为 `✅ 已完成`，并补一句结果。
- 当天未完成但仍需推进的事项，移动到「滚动跟进」。
- 从周会文档或飞书沟通新增的任务，先进入「待排期池」，再按优先级进入今日待办。
