# 第9课时：答疑课

> 课时：机动（每轮授课结束后收集问题，集中答疑）
> 定位：收集学员在学习和实际取数 / 分析中遇到的真实问题，集中解答，并沉淀成一份可复用、可检索的 FAQ。
> 说明：本课**不预设内容**，随授课动态积累。每次授课后，把学员的问题发给我，我按"问题 → 背景 → 解答 → 关联课时/资料"整理进本文档。

---

## 一、使用说明
1. **动态积累**：本课内容来自每轮授课后学员的真实提问，越用越厚。
2. **整理流程**：每次授课后 → 收集问题 → 按主题归类 → 补充解答 → 更新「高频 TOP」。
3. **学员先自查**：很多问题可能已被解答过，先查本文档 / 各课速查资料，再提问。
4. **口径优先**：涉及指标口径的问题，解答里必须写清「口径 + 公式 + 注意事项」（呼应串讲 PPT）。
