# 数据分析通识培训 - 产品同学专用

> 面向浏览器/信息流/PUSH产品同学的数据分析入门课程，所有案例均围绕内容生态负责的业务场景设计，理论+实操结合，学完即可独立完成日常取数和基础分析。

## 课程基本信息
- **总课时**：1节启动课 + 6节正课，每课时1~1.5小时
- **授课周期**：3周（每周2课时）
- **前置要求**：提前开通数据地图权限、Kyuubi查询权限

✅ **场景化**：所有案例、练习题、SQL模板均来自浏览器/PUSH/信息流真实业务需求，学完直接能用
✅ **AI友好**：全程融入AI辅助写SQL/做分析的方法，教你用提示词+校验方法，效率提升10倍
✅ **避坑导向**：总结了大量真实工作中踩过的坑，帮你少走弯路
✅ **可落地**：提供大量可直接复用的模板、Checklist，不用从零开始
