SECTION 01

为什么要做
AB 实验？

不做 AB，最容易犯三个错

🧟
幸存者偏差

只看到用了功能说好用的，没看到用了之后流失的。

🎣
归因错误

指标涨了就归功于新功能，其实可能是自然趋势 / 活动 / 节假日。

🧠
主观判断

"我觉得好""用户说想要"，上线后没人用。

AB 的核心价值：通过随机分流排除其他因素干扰，准确测量策略的真实效果。

AB 基本逻辑：唯一变量对比

对照组 A

保持原策略，什么都不改，作为对比基准。

实验组 B

上线新策略，与对照组唯一的区别就是待测策略。

把用户随机分两组，同时间同环境运行，最后对比指标差异 —— 这个差异就是策略的真实效果。

SECTION 03

必须懂的统计基础

不用背公式，理解四个概念就够

四个概念，看懂实验结果

🎲 随机波动 vs 真实效果
抛 10 次硬币不一定 5 正 5 反。即使什么都不改，两组也会有小差异 —— 要判断是真效果还是巧合。
📊 p 值
如果新策略其实没用，观察到当前差异的概率。p=0.01 = 纯靠随机出现这么大差异只有 1%。
✅ 行业标准：p < 0.05 认为显著
📏 置信区间
真实效果的可能范围，如"+3%，区间 [+1%,+5%]"。
✅ 区间不含 0 才显著
👥 样本量 / MDE
太小随机波动大易误判；太大出问题影响广。基数越大、想测的效果越小，需要样本越多。
✅ 比例类开 3~7 天，留存开 1~2 周

SECTION 04

指标体系 & 常见陷阱

三类指标，缺一不可

类型PUSH 实验信息流实验
核心指标CTR、PUSH 带来的启动 / 时长人均时长、CTR、人均 VV、有效 DAU
辅助指标到达率、展示率、点击量、落地页打开人均曝光 / 点击、完读率 / 完播率
护栏（反向）通知关闭率、卸载率、7 日留存、负反馈跳出率、APP 整体时长、卸载率
⚠️ 别只看"北极星指标"：给 PUSH 加声音震动，CTR 涨了，但用户嫌烦关通知 / 卸载 —— 护栏指标必须看。

7 大陷阱：90% 的错误都在这

陷阱 01
样本量不足 / 时间太短
开 1 天涨 10% 就上线，后面回去了 → 开够、等一整周
陷阱 02
SRM 样本比例失衡
设计 50:50 实际 55:45 → 开始就做 SRM 校验
陷阱 03
多重比较
天天看、看 10 个指标易假阳性 → 定好周期和核心指标
陷阱 04
辛普森悖论
整体涨分群都降 → 分维度看
陷阱 05
实验互相干扰
流量重叠 → 相关实验流量正交
陷阱 06
只看相对不看绝对
相对 +20% 但基准 1% → 绝对值一起看
陷阱 07
没有 AA 测试
分流本身有偏 → 先 AA 确认两组同质

SECTION 05 · 进阶

DID 双重差分法

当没法做纯随机实验时，如何剔除干扰

DID 解决两个难题

只比"组间"不行

政策后直接比政策组 vs 对照组，忽略两组原本就有的固有差异（发达地区本就增长快）。

只比"前后"也不行

只看政策组前后差异，忽略大盘时间趋势（整体上行，不改也会涨）。

DID 用"两次差分"同时解决：政策效应 =（政策后−政策前）−（对照后−对照前）
来源：内容生态产运班 · 双重差分法 DID

DID = Δ₁ − Δ₀
Δ₁ 里含了"真实效应 + 时间趋势"，Δ₀ 只含"时间趋势" —— 一减，时间趋势被抵消，剩下真实效应。

Demo · DID 如何剔除时间趋势 拖动大盘增长

假设真实政策效应 = +5（隐藏真相）

政策组：前 50 → 后55
对照组：前 40 → 后40

大盘自然增长（同时影响两组）：0

两种估计法

只看政策组前后（naive）+5
DID 双重差分+5
拖动滑块：大盘增长会让 naive 估计虚高，而 DID 始终 = +5，锁定真实效应。
来源：内容生态产运班 · 双重差分法 DID

DID 的前提假设 & 方法对比

三大前提假设

  • 平行趋势（最关键）：无干预时两组趋势平行
  • SUTVA：个体相互独立、非双边市场
  • 无遗漏变量：没有其他随时间变化的干扰
方法一句话
DID控时间趋势+固有差异，数据要求低
PSM匹配相似个体，解决选择偏差
SCM为政策组构建"合成对照"
RD断点附近近似随机

💡 DID 性价比高，常与 PSM / SCM 结合，广泛用于政策评估、增量价值测算（如 PUSH 增量）。

特别注意：双边市场的实验难题

双边市场（电商 / 外卖 / 网约车 / 社交）存在供给方 + 需求方相互依赖，普通 AB 会失效。
闲鱼发券的例子

交易物品有唯一性：给实验组买家发券促成交，会挤占对照组买家"原本能买到"的机会 —— 实验组涨、对照组被拖累，高估了效果。

怎么办

按市场 / 地域 / 时间分流（而非按用户），或用 DID / 合成控制等准实验方法评估。

来源：内容生态产运班 · 双边市场怎么设计 AB

SECTION 06

一个结论靠不靠谱？

拿到报告，按 8 条问一遍

结果可信度 · 8 条 Checklist

✓开了多久？样本量够吗？覆盖周末吗？新奇效应过了吗？
✓分流均匀吗？SRM / AA 校验过吗？两组基线一致吗？
✓核心指标提升多少？p 值？置信区间含 0 吗？
✓护栏 / 反向指标有没有负向？
✓分维度看效果一致吗？有没有辛普森悖论？
✓有没有其他同时开的实验干扰？
✓绝对值提升多少？业务价值大吗？
✓结论和建议符合数据吗？
