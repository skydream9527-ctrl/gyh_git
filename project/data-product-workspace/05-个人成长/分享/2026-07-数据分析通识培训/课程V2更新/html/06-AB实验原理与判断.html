AB 实验原理与判断

看得懂实验结论 · 判断结果可不可信

⏱️ 1.5 小时🎯 产品；销售/运营了解对照组概念即可

从"排除大盘"到"科学对照"

模块1 那句"大盘涨 10%，我涨 30%，多贡献 20%"，就是对照的朴素版。AB 实验把它做到科学严谨。

不做实验的坑后果
幸存者偏差忽略了用了就流失的人
归因错误涨了未必是功能的功劳（旺季/活动）
拍脑袋"我觉得好"，上线没人用

随机分两组，唯一区别是策略

用户流量 → 随机分流 ┬→ 对照组A（老策略）→ 收集指标 ┐ └→ 实验组B（新策略）→ 收集指标 ┴→ 对比差异

关键词是随机和同质：两组实验前必须几乎一样，否则差异不是策略造成的。

统计三件事（不背公式）

p < 0.05

是巧合的概率低于 5% → 认为显著

置信区间

不包含 0 才显著；[-1%,+3%] 含 0 = 不显著

样本量

要够（达 MDE），否则真效果也测不出

💡 p 值管"可不可信"，提升幅度和置信区间管"效果多大"——p 小 ≠ 效果大。

核心 + 辅助 + 护栏，缺一不可

类型PUSH 实验举例
核心CTR（点击/到达）、PUSH 带来的时长
辅助到达率、展示率、落地页打开率
护栏(反向)通知关闭率、卸载率、7 日留存、整体时长

⚠️ 别只看北极星：加声音震动 CTR 涨了，但用户关通知/卸载——短期赢长期输。

7 大陷阱（90% 错误在这）

  • 样本不足/时间太短
  • SRM 样本比例失衡
  • 多重比较（天天看）
  • 辛普森悖论
  • 实验互相干扰
  • 只看相对不看绝对
  • 没做 AA 测试

拿到一份实验报告，怎么读

先看问自己
可信度样本够吗？分流均匀吗（SRM/AA）？开够一个完整周了吗？
显著性核心指标 p<0.05 吗？置信区间含 0 吗？
护栏卸载 / 关通知 / 整体时长有没有被拉崩？
分维度各人群一致吗？有没有辛普森悖论？
业务值绝对提升多少？值不值得全量？

案例：PUSH 增量价值实验（DID）

对照组正常发 PUSH，实验组完全不发（关小黑屋），开 14 天。实验组用户本身更活跃 → 直接对比不准。

DID 双重差分：用【实验期两组差异】减【实验前基线差异】= PUSH 净增量。

结论：信息流 DAU +8.5%、时长 +33%、留存 +4~5%，APP 整体 DAU 几乎不变——PUSH 加深了消费深度。

🔗 这就是模块4 那个「push 价值量分析」的实验版——同一个"发 vs 不发 push"的问题，模块4 用 SQL 拆现状差异，这里用实验 + DID 科学测净增量。
