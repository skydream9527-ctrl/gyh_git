# 第2章：排序与数据过滤

> 对应原书：第 3–6 课（排序、过滤、高级过滤、通配符）
> 目标：会把结果排好序，会用 WHERE 精准筛出想要的行。

---

## 2.1 ORDER BY：排序

```sql
SELECT prod_name, prod_price FROM Products
ORDER BY prod_price;             -- 按价格升序（默认 ASC）
```
**按多列排序**（先按第一列，值相同再按第二列）：
```sql
ORDER BY prod_price, prod_name;
```
**指定方向**（DESC 降序）：
```sql
ORDER BY prod_price DESC, prod_name;   -- 价格降序，同价再按名字升序
```
> 💡 `ORDER BY` 永远放在最后。取"最贵的 5 个"：`ORDER BY prod_price DESC LIMIT 5`。

---

## 2.2 WHERE：过滤行

```sql
SELECT prod_name, prod_price FROM Products
WHERE prod_price = 3.49;
```
**WHERE 常用操作符：**

| 操作符 | 含义 | 例 |
|--------|------|-----|
| `= <> !=` | 等于/不等于 | `prod_price <> 3.49` |
| `< <= > >=` | 比较 | `prod_price >= 10` |
| `BETWEEN a AND b` | 区间（含两端） | `prod_price BETWEEN 5 AND 10` |
| `IS NULL` | 空值 | `prod_price IS NULL` |

> ⚠️ **NULL 要用 `IS NULL` 判断**，不能用 `=NULL`。而且 `WHERE prod_price <> 3.49` 一般**查不出 price 为 NULL 的行**——NULL 参与比较结果既不真也不假。

---

## 2.3 高级过滤：AND / OR / IN / NOT

```sql
-- AND：两个条件都满足
WHERE vend_id = 'DLL01' AND prod_price <= 4;
-- OR：满足其一
WHERE vend_id = 'DLL01' OR vend_id = 'BRS01';
```
> 🔴 **AND 优先级高于 OR**！混用一定加括号，否则结果全错：
```sql
-- ❌ 想要"这两家供应商里价格>=10的"，但被理解成 A 或 (B且>=10)
WHERE vend_id='DLL01' OR vend_id='BRS01' AND prod_price>=10;
-- ✅ 加括号
WHERE (vend_id='DLL01' OR vend_id='BRS01') AND prod_price>=10;
```
**IN**（在一组值里，比一串 OR 更清楚）：
```sql
WHERE vend_id IN ('DLL01','BRS01');
```
**NOT**（否定）：
```sql
WHERE vend_id NOT IN ('DLL01','BRS01');
```

---

## 2.4 用通配符过滤：LIKE

| 通配符 | 匹配 | 例 |
|--------|------|-----|
| `%` | 任意多个字符（含 0 个） | `WHERE prod_name LIKE 'Fish%'`（Fish 开头） |
| `_` | **单个**字符 | `WHERE prod_name LIKE '__ inch teddy bear'` |

```sql
WHERE prod_name LIKE '%bean bag%';   -- 包含 bean bag
```
> ⚠️ 通配符很方便但**慢**：`%` 放最前面（`'%xxx'`）无法用索引、最慢；能用普通过滤就别用 LIKE，能不把 `%` 放开头就别放。

---

## 本章要点
1. `ORDER BY` 排序，`DESC` 降序，配 `LIMIT` 取 TopN。
2. `WHERE` 过滤；NULL 用 `IS NULL`，不能 `=NULL`。
3. `AND` 比 `OR` 优先，**混用必加括号**；一组值用 `IN` 更清晰。
4. `LIKE` + `%`/`_` 做模糊匹配，但慢，`%` 别乱放开头。

## 小练习（附参考答案）
1. 取价格在 5~10 之间的产品名和价格，按价格降序。
2. 取供应商是 DLL01 或 BRS01、且价格≥4 的产品。
3. 取产品名里含 "bird" 的产品。
