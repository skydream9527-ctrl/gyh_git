# 竞品分析 Prompt 模板

> 来源：pm-workspace-starter prompts/competitor-analysis.md
> 沉淀日期：2026-06-29

## 使用方式

复制以下 Prompt，替换 `{ }` 中的内容，发给 AI 即可。

---

## Prompt

> 执行前先读 USER.md 和 MEMORY.md，已有信息（产品名、业务域、用户群）直接引用，不要重复问用户。只问缺失的信息。

我是一名产品经理，请帮我做一份竞品分析。

### 分析对象

- 我的产品：{我的产品名称}
- 竞品列表：{竞品 A}、{竞品 B}、{竞品 C}

### 分析维度

- 产品定位与目标用户
- 核心功能对比
- 最近 3 个月的版本更新重点
- 用户口碑（评分、高频差评关键词）
- 商业模式 / 变现方式
- 差异化优劣势

### 信息来源

{App Store 评论、竞品官网、行业报告、社交媒体等}

### 输出要求

1. 用对比表格呈现核心功能差异
2. SWOT 分析每个竞品
3. 标注我的产品可借鉴的功能和改进方向
4. 标注信息来源和时间
5. 对不确定的信息标注「推断」
