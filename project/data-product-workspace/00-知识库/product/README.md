## 数据产品核心能力
1. **指标体系设计**：从业务目标到可执行指标的拆解能力
2. **数据需求转化**：将业务问题转化为数据可解问题
3. **AB实验设计**：实验设计、流量分配、结果解读能力
4. **产品决策**：基于数据的产品决策能力，平衡用户价值和商业价值
5. **AI产品思维**：AI Native产品设计、Agent产品形态探索

## 工作流程
- 需求阶段：问题定义 → 指标拆解 → 方案设计
- 开发阶段：埋点设计 → 数据校验 → 实验配置
- 上线阶段：灰度发布 → 数据监控 → 效果评估
- 迭代阶段：复盘总结 → 知识沉淀 → 下一轮迭代
