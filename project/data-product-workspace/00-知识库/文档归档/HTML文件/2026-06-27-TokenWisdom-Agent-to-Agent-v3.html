Relationship Overview

TokenWisdom 是一个 User → Twin Agent → Agents 的协作系统。

用户只需要面对自己的分身 Agent；分身 Agent 代表用户参与其他工具或应用类 Agent 的讨论，并把所有关键动作带回给用户确认。

L1 Discuss

分身可加入所有 Agent 对话，代表用户提问和补充上下文。

L2 Draft

分身可生成 prompt、任务包、执行计划和权限请求。

L3 Request

分身把建议带回给用户，等待允许、拒绝或修改范围。

L4 Execute

只在用户确认后触发 Tool Agent 执行；分身本身不拥有最终确认权。
