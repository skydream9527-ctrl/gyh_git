# 浏览器 - 打点细节记录

> 来源 sheet: `细节记录`


```

短视频沉浸式
短视频沉浸式内容，在推荐列表页的expose、click事件，流的展示类型='default'，频道='推荐'；
点击短视频沉浸式内容进入沉浸式，第一个内容的play、view事件，流的展示类型='default'，频道='推荐'；
从列表页点击视频，第一个不算沉浸式（feed_channel=‘推荐’），从第二个开始才算沉浸式（feed_channel=‘沉浸视频’）
在短视频沉浸式列表页播放时，video_play事件，流的展示类型='immersion'，频道='沉浸视频'；
短视频沉浸式，从列表页点击进入详情页时，流的展示类型='related_recommendation'，频道='沉浸视频'；
短视频列表页
短视频列表页播放时，上报的expose、video_play、click等事件，流的展示类型='default'，频道='视频'；
从列表页点击进入详情页时，流的展示类型='related_recommendation'，频道='视频'；

```
