# 模板 B：周报

## 适用场景

- 个人周报
- 部门简报
- 业务线周度 Review 素材稿

## 骨架结构

```markdown
<callout color="blue">
【本周核心判断】<1-2 条，每条一句话>
【最需决策】<0-2 条待决策项>
</callout>

## 一、本周重点

### {业务/项目 A}
- **进展**：<数据 + 判断句>
- **问题**：<红色警示 + 根因>
- **下周动作**：<动作 + 负责人 + 时点>

## 二、下周计划（≤5 项）

1. <动作> / <负责人> / <时点>

## 三、风险与求助

- **风险**：<红色>
- **需决策**：<黄色背景>
```

## 填写要领

1. 每条都有判断句：不写"本周做了 X"，写"本周 X 的进展/问题说明 Y"
2. 推断必须标注 `[推断]` 前缀
3. 下周计划不超过 5 项
