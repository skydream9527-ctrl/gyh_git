# Search - 底表/宽表核心指标参考

> **数据层级**：底表/宽表（dwm/dwd 层）
> **说明**：搜索业务目前没有独立的中间表（dm/ads/dim），所有指标均基于底表/宽表查询
> **适用场景**：搜索业务的核心指标和埋点指标查询

## 概述
本文档包含搜索业务的核心指标SQL查询代码，基于dwd层底表，用于搜索业务的分析和运营监控。

## 指标说明

### 搜索核心指标说明
当前搜索核心指标文件内容为空，可能需要补充具体的搜索业务核心指标查询代码。

搜索核心指标通常包括：
- 搜索DAU（日活跃搜索用户数）
- 搜索次数（Search Count）
- 搜索点击率（CTR）
- 搜索结果点击分布
- 搜索转化率
- 搜索建议使用率
- 搜索失败率

常用的数据源可能包括：
- 搜索事件表
- 搜索结果点击表
- 搜索建议曝光表
- 搜索转化表

## 注意事项
- 搜索指标需要区分不同搜索场景（如网页搜索、图片搜索、视频搜索等）
- 需要考虑搜索质量指标（如搜索满意度、搜索成功率等）
- 搜索指标通常需要结合用户行为路径进行分析
