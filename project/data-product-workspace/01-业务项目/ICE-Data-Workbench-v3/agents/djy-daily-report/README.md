# 日报推送 Agent

面向都江堰内容池日报的专用 Agent，绑定 `djy-pool-analysis` skill。

## 能力

- 用户上传 SQL 后自动查询数据，生成 AI 日报并按确认结果推送飞书。
- 完整日报生成、补跑、推送前检查、推送后摘要。
- 4 家 CP 内容池字段校验和异常解释。
- DAU 消费指标趋势、实验组/CP/频道维度分析。
- 创作者签约审核链路一致性校验。

## 典型输入

- “补跑 20260531 都江堰内容池日报，只生成报告不推送。”
- “我上传了 3 个 SQL，帮我跑数并生成今天的 AI 日报，确认后推送飞书。”
- “把昨天都江堰日报推到飞书群，先给我确认摘要。”
- “查一下贝壳 CP publish_time 异常明细。”
- “看下这批签约作者有没有推到审核表并入内容池。”

## 依赖

- `skills/djy-pool-analysis`
- Kyuubi/DataWorks 查询环境
- 飞书发布/图片上传能力
- HTML/PNG 生成环境
