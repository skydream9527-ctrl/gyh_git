📐 v4 设计决策（已锁定）

  • 底色：默认 light（纸张白 #faf9f6 + 侧栏 #f4f1ec + 纯白卡片），右上 🌙/☀️ 切 dark · localStorage 跨页持久化。
  • Accent：单一克制蓝 #1a73e8，全部 CTA / 链接 / 选中态同色，不用渐变。
  • 范式色：5 个低饱和 chip（蓝/琥珀/紫/绿/红），仅作分类标识。
  • 形状：按钮全部胶囊形 border-radius:99px，卡片 14 px，头像 100% 圆。
  • 三栏：Sources 280 / Chat 1fr / Studio 340（已确认）。
  • Citation：行内 [1] [2] chip + 下方面板默认展开第 1 条，2/3 折叠在 ▾ 后（已实装于 02）。
  • 笔记本卡：保留 emoji 作主图标（已确认）。
  • 移动端 Workspace：3 栏折叠为顶部三段切换 (数据源 · 对话 · Studio)（已确认）。
  • 字体：Inter 正文（500 字重做标题）+ PingFang SC 中文兜底 + JetBrains Mono 代码。

🎨 暗夜模式实现细节

  • 每个 HTML 在 <head> 内置一段 IIFE，渲染前读 localStorage.getItem('ice-theme') 应用 data-theme 属性，避免 FOUC（白闪）。
  • 顶栏图标按钮：light 显示 🌙 (点击切到 dark)，dark 显示 ☀️ (点击切到 light)。
  • 所有色 token 走 :root[data-theme="dark"]，没有硬编码颜色（已修 alert / 浏览器 chrome 等边缘场景）。
