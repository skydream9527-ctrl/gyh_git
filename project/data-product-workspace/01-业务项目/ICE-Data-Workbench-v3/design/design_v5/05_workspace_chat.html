📐 这张稿的布局重设计

  • 原版：三栏 grid 280 / 1fr / 320，纯白底 + 蓝 accent，左右栏中性灰，工具卡浅色块。
  • v5 三栏：280 / 1fr / 340 — 左右栏改用 surface-2 暖米色（区分主体白底中栏），中栏 bg = 暖米白纸张色，工具卡 = 暖墨深色块 #1a1612（终端感，对比强烈）
  • 顶栏：54px workspace mode + 面包屑（Dashboard / 任务名 + AB chip），右侧叠层协作者头像（边框白底显层级）+ 分享按钮 + ⋯ 菜单
  • 左栏 · 工作文件：分组卡（SQL / 数据 / 输出），可折叠 ▾▸，每组 count badge 用 Mono 字。文件行 hover 才出现 📎 / × 操作。Mono 大写 section 标题 + 横线引导符
  • 左栏 · 知识库：紧凑 list，飞书 wiki 用 🪶 图标，文档数 badge 用 Mono
  • 中栏 · 对话气泡：User 用铜橙实色 + 阴影 + 圆角 14/14/4/14（右下角削角）；Assistant 用白底 + 32px 范式色头像 + 圆角 14/14/14/4（左下角削角，对称镜像）
  • 引用 chip：行内 [1] [2] Mono 字 + accent-soft 底，hover 实色铜橙
  • Tool Call 卡：嵌在 assistant 气泡内的暖墨终端块，3 段（head 工具名 + 状态 / body 代码或参数 / foot 耗时 + 行数），状态用半透明色块（绿✓/橙⏵/红✗）
  • 消息 meta：Mono 大写 + 横线，显示时间 + 模型 + 耗时
  • ChatInput：surface 白底 + 14px 圆角，focus 铜橙 ring。引用 chip 区在 textarea 上方（kb-ref-chips，kb 内容用 Mono）。底部操作栏：@ 引用 / 计划模式 / 在线（小 chip 按钮）+ 发送提示 Enter / Shift+Enter（Mono 小字）+ 🎙 圆形语音 + 铜橙实色发送按钮带 ↵ kbd
  • 右栏 · 4 Tab：对话 / 定时 / Skill / Agent，active tab 铜橙下划线（学 GitHub）。当前展示 Agent tab：Agent 卡 + 文件清单 + 绑定 Skill chips（含 cat 标识）+ 经验卡片列表
  • 移动端：单栏，顶 task-name + 任务 meta，顶部 3 段切换（📂 文件 / 💬 对话 / 📋 详情，segmented control 风），消息流单列（user 气泡占近全宽），输入框 textarea font-size 16 防 iOS zoom

和 v4 / 当前实现的差异

  • 底色：纯白 / 灰 → 暖米白 + 暖米色 + 暖墨深色（工具卡）三层
  • 气泡：扁平彩色 → User 铜橙实色 + 立体阴影；Assistant 白底 + 范式色头像 + 镜像圆角
  • 工具卡：浅色块 → 暖墨深色终端块（Claude Code 风），含语法高亮
  • 引用 chip：[1] 普通 chip → Mono 字 + accent-soft chip，hover 反白
  • Tab 风格：底部下划线 → 保留下划线但用铜橙单色 + 字体 12px Mono 风（更克制）
  • 右栏 Agent：原版纯文本 → Agent 卡 + 文件分组 + Skill chips + 经验卡片列表（结构化）
