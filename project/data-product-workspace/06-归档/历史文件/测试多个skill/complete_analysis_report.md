## 七、附录

### 7.1 数据来源

- **数据查询时间**：2026-04-07
- **数据查询工具**：小米数据工场（DataWorks）
- **分析工具**：Python + Pandas

### 7.2 指标说明

| 指标 | 说明 | 类型 |
|------|------|------|
| DAU | 日活跃用户数 | 计数 |
| 有效率 | 有效用户占比（曝光≥6或点击≥1） | 比率 |
| 人均曝光 | 平均每人每天的信息流曝光次数 | 均值 |
| 人均VV | 平均每人每天的视频播放次数 | 均值 |
| 人均信息流时长 | 平均每人每天的信息流使用时长（分钟） | 均值 |
| 人均消费时长 | 平均每人每天的内容消费时长（分钟） | 均值 |
| CTR | 点击率（点击次数/曝光次数） | 比率 |
| UTR | 用户点击率（有点击的用户数/总用户数） | 比率 |
| 曝光到曝光留存 | 次日仍有曝光的用户占比 | 比率 |
| 曝光到有效留存 | 次日成为有效用户的占比 | 比率 |
| 有效到有效留存 | 次日仍为有效用户的占比 | 比率 |

### 7.3 显著性标记说明

- ✅ **显著**：相对变化 > 5% 或 p < 0.05
- ⚠️ **边缘显著**：相对变化 3%-5% 或 0.05 ≤ p < 0.10
- ➡️ **不显著**：相对变化 < 3% 或 p ≥ 0.10
