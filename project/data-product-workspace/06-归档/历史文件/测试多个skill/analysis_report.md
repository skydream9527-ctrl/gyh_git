</callout>

<callout emoji="warning" background-color="light-yellow">
**⚠️ 需关注指标**

**大盘用户 & 老用户：**
- 人均信息流时长下降：-2.91% ~ -5.43%
- 人均消费时长下降：-2.07% ~ -4.67%
- UTR略有下降：-0.76% ~ -1.22%

**新用户：**
- 数据波动较大，需持续观察
</callout>

### 5.2 发布建议

<callout emoji="rocket" background-color="light-blue">
**综合评估：建议继续灰度发布**

**理由：**
1. **核心规模指标正向**：DAU显著提升，用户规模扩大
2. **消费质量指标正向**：人均VV提升，CTR提升，说明内容消费质量改善
3. **新用户表现优异**：新用户的各项指标均有显著提升，说明新版本对新用户更友好
4. **留存稳定**：大盘和老用户的留存指标基本持平，新用户留存有提升趋势

**需关注的问题：**
1. 人均消费时长下降，需分析是否为内容类型变化导致
2. 部分SQL查询失败，建议补充完整数据后再做全面评估
</callout>

### 5.3 后续行动建议

### 6.2 指标说明

| 指标 | 说明 |
|------|------|
| DAU | 日活跃用户数 |
| 有效率 | 有效用户占比（曝光≥6或点击≥1） |
| 人均曝光 | 平均每人每天的信息流曝光次数 |
| 人均VV | 平均每人每天的视频播放次数 |
| 人均信息流时长 | 平均每人每天的信息流使用时长（分钟） |
| 人均消费时长 | 平均每人每天的内容消费时长（分钟） |
| CTR | 点击率（点击次数/曝光次数） |
| UTR | 用户点击率（有点击的用户数/总用户数） |
| 曝光到曝光留存 | 次日仍有曝光的用户占比 |
| 曝光到有效留存 | 次日成为有效用户的占比 |
| 有效到有效留存 | 次日仍为有效用户的占比 |
