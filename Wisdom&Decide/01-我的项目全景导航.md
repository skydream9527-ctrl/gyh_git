## 五、每日工作流快速导航

| 场景 | 去哪里找 / 做什么 |
|-----|------------------|
| 要做一个新决策 | 先翻 Wisdom&Decide → 用 decision-frameworks 工具 → 记录到决策日志 |
| 设计一个新 Agent 功能 | 先看 project/agent-llm/agents/ → 对比现有 Agent 实现 → 参考 design_decisions.md |
| 做 AB 实验 | project/ab-testing/ → 检查样本量 → 实验后用因果推断方法解读 |
| 指标异动归因 | project/causal-inference/ → 拆解维度 → 用 DID/PSM 等方法验证 |
| 写 PRD / 方案 | 先看现有 design_decisions.md → 覆盖异常流程 → 用表格对比方案 |
| 日常开会/沟通 | 记录到 data-product-workspace/04-会议纪要/ |
| 每周/季度复盘 | 更新 data-product-workspace/05-个人成长/季度复盘/ |
| 开发新功能 | backend/ + frontend/ 找现有模块参考 → 先加测试再实现 |
