# 实验组映射关系

## 核心字段：`exp_group_6`

都江堰数据统一使用 36% 流量组进行对比，对应字段为 `exp_group_6`（model.sql 第 7-14 行）：

| exp_group_6 值 | 含义 | 来源逻辑 |
|---|---|---|
| `36%自建组` | 自建推荐流量 | exp_group = '36%自建组' |
| `36%火山组` | 火山推荐流量 | exp_group = '36%火山组' |
| `自建反转组` | 自建流量反转验证 | observation_group = '2%自建反转组' |
| `火山反转组` | 火山流量反转验证 | observation_group = '2%火山反转组' |
| `自建纯净组` | 自建纯净对照 | observation_group = '2%自建纯净组' |
| `火山纯净组` | 火山纯净对照 | observation_group = '2%火山纯净组' |

## 日常分析用法

**最常见对比**：`36%自建组` vs `36%火山组`

```sql
WHERE exp_group_6 IN ('36%自建组', '36%火山组')
GROUP BY exp_group_6
```
