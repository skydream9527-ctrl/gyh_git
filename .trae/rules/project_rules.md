# PM 工作区规则

> 本文件由 Trae IDE 在每次对话开始时自动注入。

---

## 启动顺序

每次对话开始，AI 必须按顺序读取：

1. AGENTS.md — 行为规则和踩坑记录
2. MEMORY.md — 长期记忆索引
3. USER.md — 个人画像

同一 session 内已读过的文件不重复读取。

---

## 核心规则

### 数据引用规范

引用市场数据必须标注来源和时间；无数据支撑的判断必须标注「推断」。

### 飞书文档操作

涉及飞书文档写入/创建时：
- 先确认目标文档是否已存在
- 有则用 doc_append / doc_insert 更新
- 无则新建
- 禁止在非重建场景使用 doc_write

### 文件写入规范

任何有明确输出位置的任务（PRD、竞品分析、决策记录），优先写入对应文件，再告知用户路径。

---

## 记忆更新规则

每次对话结束时，AI 必须检查并执行：

1. IF 对话中产生了新的决策 → 追加到 MEMORY.md「关键决策」段
2. IF 对话中发现了新的偏好 → 追加到 AGENTS.md「踩坑记录」段
3. IF 对话中有值得记录的内容 → 写入 memory/YYYY-MM-DD.md
4. IF 对话中犯了错误 → 追加到 AGENTS.md「踩坑记录」段，标注「教训」
