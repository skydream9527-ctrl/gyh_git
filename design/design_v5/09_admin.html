📐 这张稿的布局重设计

  • 原版：TopNav admin mode + 双栏（左 240 sidebar / 右 main），sidebar 4 组导航（监控 / 内容审核 / 资源管理 / 用户与配置），main 是 AdminOverview（page-head + 待处理事项 + 4 stat + 双栏：最近用户 / Agent 排行）
  • v5：沿用双栏 layout（左 240 / 右 1fr），但视觉全面升级。TopNav 加 🛡 ADMIN pill（紫 muted 范式色 mono 字）凸显当前在管理后台
  • 左侧栏：暖米色 surface-2 底，分组用 Mono 大写小字 + 横线引导符（CONTENTS 风），导航项 hover surface 白底，active 用铜橙 soft + 600 字重，待处理项右侧 红色 Mono badge（14 / 6 / 3 / 7）。底部 footer 显示版本号 + uptime + ALL SYSTEMS GREEN（绿色 dot）—— 学 Anthropic Console / Vercel Dashboard 风
  • 主内容 page-head：Mono tag「ADMIN · OVERVIEW · 监控」+ 36px Inter H1「📊 概览」+ 副本
  • 🚨 待处理事项卡：warning 软色底 + 3px 左边色条，标题 + 右上 Mono 计数（15 ITEMS），内部行用范式色左边条 + Mono CTA 「→ 调整预算 / 立即审批」（铜橙）。预算超出用 error 色突出
  • 4 stat 卡居中风（与 08_scheduled 一致）：每张含 44px icon（👥/📋/💬/💰）+ Mono label + Inter 32px 数字 + 趋势。每张不同范式色 ico（accent / biz / data / warning）。预算超出态用 error 色 icon + ⚠ down 趋势
  • 双栏 1.1 : 1：左「最近注册用户」（小头像彩色渐变 + 名字 + role chip USER/ADMIN/SUPER + 邮箱 mono + 时间 mono）/ 右「Agent 排行」（#1-#5，#1-#3 用铜橙实色圆 + Inter 名字 + Mono 消息数 + 满意度 + 铜橙实色进度条）
  • section 头：Inter H3 + 右 Mono 大写 meta（7 NEW / LAST 30 DAYS）
  • 移动端：左 ☰ 抽屉触发按钮 + 中 ADMIN pill + 右头像。主内容单栏（精简 alert + 2×2 stats + 紧凑用户列表 + Agent 排行 top 2）

和 v4 / 当前实现的差异

  • TopNav：纯 breadcrumb → 加 🛡 ADMIN pill 紫范式色 凸显环境
  • 侧栏：浅色 + 单色 active → 暖米色 + Mono 分组小字 + 铜橙 active 高亮 + 红 badge
  • 侧栏底部：原版无 → 新增版本号 / uptime / 系统状态
  • page-head：emoji + 单行 → Mono tag + Inter 大标题 + 副本
  • alert：彩色块 + 行 → warning 软色底 + 范式色左边条 + Mono CTA
  • 4 stat：左对齐数字 + 标签 → 居中风 + icon + Inter 数字 + Mono 趋势
  • 用户列表：纯文字 → 彩色渐变小头像 + role chip + Mono 邮箱
  • Agent 排行：纯 #编号 + 进度条 → top 3 铜橙实色圆 + Inter 名字 + Mono 数字 + 铜橙实色填充条
