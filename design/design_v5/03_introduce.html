01 / Core concepts

六个概念，
构成一个工作台。

从一次对话到一份报告，所有数据工作流都用这六个概念组合而成。

📋
任务
一次完整的数据工作流，独立工作空间，记录对话与产出。
🤖
Agent
公共智能体，与范式 1:1 绑定，越用越聪明。
⚡
Skill
可执行工具，Agent 通过 function calling 调用。
✨
经验卡片
对话中提炼的规则，审批后注入 Agent 上下文。
🎯
工作范式
5 种范式预绑定 Agent + Skills + Prompt。
🌐
公共区
团队共享的 Agents、Skills、文件、模板。

数据分析

浪潮归因

03 / Full feature coverage

不止聊天，
覆盖完整工程。

从认证到存储，从 Agent 进化到定时调度，工作台的每一层都为长跑而设计。

01
认证与三级角色
飞书 OAuth + JWT 双 token + super / admin / user 三级权限。
02
Workspace 三栏
文件 + 流式对话 + Tool Calling + 文件预览。
03
Agent 进化
Prompt 版本历史 + 测试沙盒 + 经验卡片注入。
04
文件优先存储
文件系统是 source of truth，SQLite 仅 cache。
05
定时任务
cron + Agent 自动执行 + 飞书 / 邮件推送。
06
管理后台
用户 / Agent / Skill / KB / 审计 / 用量监控全覆盖。
