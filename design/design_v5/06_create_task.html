📐 这张稿的布局重设计

  • 原版：max-width 880 居中 + ct-stepper 横向（仅圆点 + 横线 + 文字标签 + active 主色）+ ct-section（白底卡 padding 24）+ 3 步切换。
  • v5：max-width 920 + 增加大 Hero 区（Mono 大写 tag「CREATE A NEW TASK」+ 48px 衬线 H1「从一个 Agent 起步」+ 副标题）—— 让创建流程不再是冷冰冰的表单页
  • Stepper：从「圆点 + 单行 label」 → 印刷感横向（32px 数字圆 + Inter step 名 + Mono 大写副标，中间 56px 短分隔线，已完成时填铜橙）—— 学 Stripe Dashboard 的 onboarding 步骤条
  • Section 头：左 H2 衬线标题 + 副标题 / 右 Mono 大写 STEP 01 / 03 进度标识（border-bottom 分隔）
  • Step 1 · Agent 网格：auto-fit minmax(200, 1fr) 卡片 height 170 + 3px 范式色 border-top + Mono 大写 meta（如 DATA · v2.1.0）+ 选中态 2px box-shadow ring + 右上角 ✓ 圆形勾选标识。分组标题用 Mono 大写「已上线 · 6 / 即将上线 · 2」+ 横线引导符
  • Step 2 · 表单：grid 1fr 1fr，input height 40 + 8px 圆角 + focus 铜橙 ring，textarea 等高自适应，hint 用 Mono 小字
  • Step 3 · 折叠：3 个 <details> 卡（⏱ 定时 / 🌐 可见性 / 等），summary 用 Inter 小标题 + 右侧 Mono ▾▸ 箭头，open 时上分隔线 + surface-2 头底
  • cron preset：4 个 Mono 字按钮（每天 09:00 / 工作日 09:00 / 每周一 09:00 / 每小时），active 状态铜橙 soft
  • 底部 action bar：左 Mono 进度提示「已选择 📊 数据分析 · 1 / 3」/ 右 三按钮（取消 / 上一步 / 下一步），border-top 分隔
  • 移动端：单列。Stepper 改成横滚 pill（圆点 + 文字 inline，active 铜橙 soft），content 单列 agent grid 2 列，主操作 sticky 底部

和 v4 / 当前实现的差异

  • 布局：纯表单页 → Hero + Stepper + Section（增加叙事层）
  • Stepper：圆点 + 横线 + label → 32px 数字 + Inter step 名 + Mono 副标 + 短分隔线
  • Agent 卡：保留尺寸（170 高 / 3px 边）→ 文字字体改 Inter / Mono，新增选中态 ✓ 圆形角标
  • Section 头：单行 h3 → 左 H2 衬线 + 副标 / 右 STEP 01 / 03
  • cron preset：纯文字按钮 → Mono 等宽字按钮
  • 底部：right-aligned 按钮 → 左进度 + 右按钮（带选择回显）
