CORE FEATURES

分析的全流程，被重新组织

不再在 BI / Excel / IM / 文档之间反复切换。一切围绕「笔记本」展开。

📒

笔记本式分析

每个分析任务一个笔记本，对话、数据源、生成产物同屏。引用 [n] 可点击溯源。

🤖

专属 Agent

面向 5 个范式调优的 Agent，调用各自 Skill 集（指标拆解、归因、SQL、图表）。

🎨

Studio 沉淀

音频简报 / 经验卡片 / 归因脑图 / 完整 PDF 报告，一键生成可分享。

📚

知识库联动

飞书 / Mify / 内部 Wiki 实时同步，RAG 检索，规范口径自动加载到上下文。

⏱

定时与告警

把笔记本变成跑批：日报、周报、监控告警自动推 IM，结论自动写卡片。

👥

协同与审批

多人共编、@ 提及、@@评论、经验卡片审批流程，知识沉淀有归属。
