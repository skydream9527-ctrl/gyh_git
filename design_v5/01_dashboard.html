📐 这张稿的核心决策

  • 顶栏：56 px sticky，backdrop-filter blur(16px) + 70% 透明背景，导航 inline 排在 brand 右侧（不再用左侧栏，桌面端导航走顶栏）
  • 问候语："早上好，Yunhe 👋" + 当日待办计数（v4 没有这一层叙事，v5 让首页有人情味）
  • Stat 卡：4 列等宽，每张右下挂 spark line SVG（饱和度 18%），数字 tabular-nums
  • 范式区块：升级为主入口 —— 顶部柔性 radial-gradient 强调，5 张卡平铺，CTA 按钮换成渐变主按钮放右上
  • 双栏：1.6 : 1（左任务列表 / 右团队动态），任务列表带 chip + 状态 + 时间三列，hover 出 surface-2 底
  • 动态时间轴：左侧头像 + 32 px 圆点 + 1px 连接线，模仿 GitHub Discussions 风
  • 手机端：顶栏精简到 brand + 3 icon，问候语 22px，stat 卡 2 × 2 网格，范式横向滚动，FAB 浮动右下，底栏 4 Tab 玻璃模糊
  • 暗夜模式：所有色全部走 token，顶栏背景 rgba(13,14,18,.7)，spark 线条更亮
