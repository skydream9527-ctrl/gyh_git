📐 v5 设计决策

  • 底色：light #ffffff（纯白，Stripe 风）/ dark #0d0e12（Cursor 深墨）。 不再用 v4 的纸张色。
  • Accent：#635bff → #8b85ff 渐变紫，主按钮 + brand mark + key CTA 用渐变；其余链接 / 链状态用纯色 #635bff。
  • 形状：圆角分级 — 按钮 8 / 卡片 12 / 大卡片 16 / chip 6 / pill 999。不再全胶囊。
  • 阴影：Stripe 招牌「微阴影 + 边框」叠加 — 卡片同时有 1px border 和 shadow-1，避免纯靠阴影。
  • 字体：Inter（cv11/ss01）+ JetBrains Mono；数字一律 tabular-nums。中文 PingFang SC 兜底。
  • 范式 5 色：indigo / orange / violet / cyan / pink，饱和度统一在 60–70%，dark 模式下 desaturate 到 50%。
  • 顶栏：56 px sticky，背景 70% 透明 + backdrop-filter blur 16px（Stripe 经典）。
  • 键盘优先：⌘K 搜索、G + 字母 跳页、N 新建。所有 .kbd 提示组件已就位。
  • 暗夜模式：渲染前 IIFE 读 localStorage('ice-v5-theme')，避免 FOUC。

🆚 v4 → v5 关键变化

  • 底色：纸张白 #faf9f6 → 纯白 #ffffff；侧栏纸张色去掉
  • Accent：单色蓝 #1a73e8 → 渐变紫 #635bff→#8b85ff
  • 形状：全胶囊（99px） → 圆角分级（8/12/16）
  • 阴影：极弱（border 主导） → Stripe 风（border + shadow 双层）
  • 字体：Inter 500 字重 → Inter 600 字重 + 更紧字距 letter-spacing
  • 范式色：低饱和（NotebookLM 克制） → 中饱和（更彩，更现代）
  • 三栏：Sources/Chat/Studio → 见 Workspace 稿（保留三栏，但视觉重新调度）
