设计决策

  • 底色：暖米白纸张 #faf4ed（light）/ 暖墨深底 #1f1d1a（dark）。抛弃纯白和纯黑，所有色都偏暖。
  • Accent：Claude 铜橙 #cc785c 实色。没有渐变。仅用于 brand mark / primary 按钮 / 关键 link。
  • 字体：Fraunces（衬线）做所有标题 + 起点卡名 + 任务名 + 范式名 + brand。Inter 做正文。JetBrains Mono 做 chip / 时间 / 元数据 / 数字。
  • 形状：圆角分 4 / 6 / 8 / 10 / 11 / 999。卡片统一 10-11px，按钮 8px，chip 4px。没有胶囊按钮。
  • 阴影：极弱（shadow-1 = 1px 0 4% alpha），主要靠边框和留白。
  • 范式 5 色：sage / terracotta / muted-purple / steel / rose，都是暖底低饱和。
  • Chip：用 mono 大写字母（AB / BIZ / DATA），不用中文 chip 文字 —— 学 Claude Code 文档的 01 / 02 / 03 编号风。

v4 → v5 关键变化（NotebookLM → Claude Code）

  • 底色：#faf9f6（凉米白） → #faf4ed（暖米白）— 加重了纸张感
  • Accent：#1a73e8（蓝） → #cc785c（铜橙） — 从冷转暖
  • 字体：Inter 全场 → Fraunces 衬线标题 + Inter 正文 + Mono chip — 引入字体对比
  • 形状：全胶囊（99px） → 圆角分级（4/6/8/10/11） — 砍掉胶囊
  • chip：低饱和 NotebookLM 5 色 → 暖调低饱和 5 色 + Mono 大写文字
  • 顶栏：blur 半透 → 实色边框（更克制 / 印刷品感）
  • 布局尺寸：完全沿用真实代码（dash-main 1200max / paradigm 160h / task 132min-h）
