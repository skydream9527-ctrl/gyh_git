Understanding Paradigms

ICE Workbench abstracts data workflows into 5 paradigms. Each paradigm pre-binds Agent + Skills + Prompt. You always: "Select Paradigm → Describe Problem → Agent Concludes".

ParadigmScenarioAgent
dataSQL · Visualizationdata-analysis
abA/B test significanceab-experiment
bizMetric drill-downbiz-insight

Creating Your First Task

A task is the atomic unit of ICE Workbench —— a complete data workflow including conversation, files, and outputs.

Three Entry Points

  1. From Dashboard click + New Task
  2. From Dashboard "Quick Start" click a paradigm card
  3. Scheduled tasks create a task automatically
# Example: Create a task with data paradigm POST /api/v1/tasks { "name": "Double 11 A/B Review", "paradigm": "ab", "agent_id": "ab-experiment" }

Workspace Layout

The workspace features a 3-column layout: Left Files · Center Chat · Right Preview.
