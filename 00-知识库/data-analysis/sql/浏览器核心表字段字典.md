|------|------|------|----------|
| `iceberg_zjyprc_hadoop.browser.dwm_browser_event_aggregation_miui_di` | ICEBERG | did+date | APP启动、版本、启动方式、整体使用时长、基础行为数据 |

## 一、dwm_browser_event_aggregation_miui_di （MIUI端用户行为聚合表）

### 1. 维度字段（✅ 已验证）

| 字段名 | 类型 | 说明 | 枚举值/使用说明 |
|--------|------|------|-----------------|
| `date` | integer | 日期 | 格式 yyyyMMdd，如 20260628 |
| `did` | varchar | 设备ID | 用户唯一标识，去重计算DAU使用 |
| `app_ver` | varchar | APP版本号 | 格式：`主版本.次版本.构建号`，如 20.21.1050617 |
| `os_ver` | varchar | 操作系统版本 | - |
| `model` | varchar | 机型 | 可用于高端机/低端机分群 |
| `app_launch_way` | varchar | 启动方式 | ✅ 已确认完整枚举值（按用户量排序）：<br>• `点击icon`（主启，~38%用户）<br>• `第三方调起`（三方唤起，~36%）<br>• `点击push`（Push通知启动，~10%）<br>• `新全搜调起`（全局搜索，~3%）<br>• `newhome搜索框`（桌面搜索框，~1%）<br>• `负一屏`（桌面负一屏，~1%）<br>• `newhome搜索调起`（桌面搜索调起，~0.4%）<br>• 其他小流量入口：widget各类、点击桌面书签、calendar、subscribe_push等 |
| `third_packagename` | varchar | 第三方调起来源包名 | ✅ 仅当 `app_launch_way='第三方调起'` 时有值<br>Top渠道见下文「三方渠道枚举」 |
| `is_app_dau_2024` | integer | APP DAU标记 | =1 表示当日是APP活跃用户，建议过滤条件加上此字段保证数据准确性 |
| `is_new_2024` | integer | 新用户标记 | =1 表示当日是新用户 |

> ⚠️ 注意：`app_packagename` 字段全为NULL，无法通过包名区分版本类型

### 2. 指标字段

| 字段名 | 类型 | 说明 | 单位 |
|--------|------|------|------|
| `app_open_cnt` | bigint | APP打开次数 | 次 |
| `app_dura` | bigint | APP总使用时长 | 毫秒 |
| `app_duration_cnt` | bigint | 有使用时长的记录数 | - |

## 四、常用查询模板

### 1. DAU&基础指标查询模板
```sql
SELECT
    date,
    COUNT(DISTINCT did) AS dau,
    SUM(app_open_cnt) / COUNT(DISTINCT did) AS avg_open_cnt,
    ROUND(SUM(app_dura) / COUNT(DISTINCT did) / 60000, 2) AS avg_app_duration_min
FROM iceberg_zjyprc_hadoop.browser.dwm_browser_event_aggregation_miui_di
WHERE date BETWEEN ${start_date} AND ${end_date}
  AND app_open_cnt + app_duration_cnt > 0
  AND COALESCE(did, '') != ''
  AND is_app_dau_2024 = 1
GROUP BY date
ORDER BY date;
```

## 五、⚠️ 已知注意事项

1. **过滤条件**：查询活跃用户务必加上 `is_app_dau_2024 = 1`，排除异常数据
2. **空值处理**：did字段需要过滤空值 `COALESCE(did, '') != ''`
3. **置顶内容**：计算自然信息流指标，要过滤 `is_top = 0` 排除置顶内容
4. **版本类型**：核心表中没有直接的版本类型字段（默认版/简洁版/高端版），需要通过版本号段判断（待确认规则）
5. **三方渠道空值**：`third_packagename` 会有 `homeAppId_extAppId_null` 和空值，分析有效三方渠道建议过滤掉这两类
6. **单位转换**：时长类字段（app_dura, feed_dura等）都是**毫秒**，转分钟需要除以 60000
