## 🔍 新口径说明
本次分析基于神策新埋点规范：
| 指标 | 口径 |
|------|------|
| UV | `page_expo_view_new`（页面曝光）+ `content_duration_new`（时长上报）两事件的 DISTINCT_ID 去重合集 |
| 时长 | `content_duration_new` 事件的 `duration` 参数，单位毫秒，转换为分钟 |
| 过滤条件 | 仅浏览器主端，小说/短剧保持官方exp_id过滤 |

> ⚠️ **重要踩坑**：不能仅从`page_expo_view_new`统计UV，因为小说阅读页、沉浸式视频、短剧等场景只有`content_duration_new`时长事件，没有曝光事件，会导致UV被低估、人均时长出现几百分钟的异常值。

## 📊 核心结论
1. **推荐频道是绝对基本盘**：日均UV 788万，占信息流核心流量65%以上，人均时长15.2分钟，频道次日留存41.2%，远高于所有垂类
2. **Push是第二大高质量流量入口**：日均UV 233万，人均时长11.1分钟，频道次日留存30.2%，是除推荐外留存最高的流量来源
3. **热点流量大但粘性差**：日均UV 140万，人均时长12.9分钟，但频道次日留存仅21.2%，APP留存与频道留存差29.6pp，用户"看完即走"
4. **沉浸式内容消费深度最高**：沉浸式视频页人均31.2分钟、小说阅读页人均34.6分钟、短剧详情页人均24.1分钟，是普通列表页的3-5倍
5. **垂类频道普遍未形成用户习惯**：视频/短剧/热榜/小视频等频道次日留存均在10%以下，底tab小视频仅4.9%
6. **小说/短剧高价值但规模小**：人均时长分别达到34.6/22.9分钟，但日均UV仅20-30万，属于高价值小众场景

## 🛠️ 方法沉淀
### 神策新埋点使用注意事项
1. UV统计必须同时包含曝光事件和时长事件，取用户并集去重
2. `content_duration_new.duration`是字符串类型，需要CAST转换为数值类型
3. 时长单位是毫秒，需要除以60000转换为分钟
4. 部分页面没有`feed_channel`字段，需要用`COALESCE`处理为"未知"
5. 需要过滤掉`feed_minivideo_continuously`页面下`feed_channel = '视频底tab小视频'`的异常数据

### 批量查询执行方法
参考`/tmp/execute_new_caliber.py`脚本：
- 使用Python subprocess调用kyuubi CLI
- 支持跳过已存在的结果文件，断点续跑
- JSON格式保存结果，方便后续处理
