# 数据分析 Prompt 模板

> 来源：pm-workspace-starter prompts/data-analysis.md
> 沉淀日期：2026-06-29

## 使用方式

复制以下 Prompt，替换 `{ }` 中的内容，发给 AI 即可。如有数据平台表结构，一并提供效果更好。

---

## Prompt

> 执行前先读 USER.md 和 MEMORY.md，已有信息（核心指标、数据平台、当前项目）直接引用，不要重复问用户。只问缺失的信息。

我是一名产品经理，请帮我做以下数据分析。

### 分析目标

{想回答什么业务问题？例如：为什么本周留存率下降了？}

### 数据说明

{提供表名、字段说明、时间范围}

### 分析维度

- {维度 1：例如按用户群拆分}
- {维度 2：例如按时间趋势}

### 输出要求

1. 如果需要 SQL，请写出可执行的 SQL 查询语句
2. 用表格呈现分析结果
3. 对数据变化给出解读（标注「数据事实」和「推断」）
4. 如果数据不足以支撑结论，明确指出
5. 给出下一步建议（是否需要更多数据、是否需要 A/B 实验）
