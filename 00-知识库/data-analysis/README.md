## 业务覆盖
- **浏览器主端**：DAU、留存、使用时长、功能渗透率
- **信息流**：点击率、停留时长、互动率、内容消费指标
- **内容中心**：内容供给、分发效率、创作者生态
- **搜索**：搜索量、点击率、满意度、零结果率
- **小说**：阅读时长、留存、付费转化

## 核心能力
- 指标异动排查方法论
- AB实验结果解读
- 用户行为路径分析
- 业务问题根因定位
- 自动化分析脚本开发
- AI辅助分析提效
