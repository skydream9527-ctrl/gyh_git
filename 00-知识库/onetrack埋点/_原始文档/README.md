## 复现知识库

如需从原始 Excel 重新生成知识库文档与索引:

```bash
# 1. 解析两份 Excel 为结构化 JSON
python3 ../_scripts/parse_onetrack.py

# 2. 生成 Markdown 文档 + JSON 索引
python3 ../_scripts/build_onetrack_docs.py
```

> 脚本内默认读取 `/Users/mi/Downloads/` 下的源文件。若源文件已移动,请修改 `_scripts/parse_onetrack.py` 顶部的 `files` 列表指向本存档目录,或重新指定路径。

## 注意事项

- 两份 Excel 均为带格式的 .xlsx(含合并单元格、字体颜色标记)。内容中心的字体颜色有版本语义:黑色=一期已打、紫色=二期需打、红色=二期重点、蓝色=三期重点、黄色底=新增字段。解析脚本已提取蓝色/红色/黄色标记到参数的 `color_tag` 字段。
- `浏览器新版OneTrack埋点汇总.xlsx` 体积较大(10.3 MB),主要因 `ad商业化事件`、`AI搜索` 等 sheet 含大量参数与多行文本说明。
- 原始 Excel 若有版本更新,建议保留旧版存档(按日期后缀命名,如 `...xlsx.20260701`),并重新运行解析脚本刷新知识库。
