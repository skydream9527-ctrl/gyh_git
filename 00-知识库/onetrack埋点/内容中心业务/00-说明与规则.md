# 内容中心 - 说明与打点规则

本文件汇总打点命名规则、版本颜色图例及未启用模块说明,便于回溯原始文档。


---

## 打点规则


```

打点规则 | 事件名（英文） | 事件名（中文） | 属性名（英文） | 属性名（中文） | 属性值类型 | 属性名的对应值及说明 | 备注
event name（English） | event name（Chinese） | key name（English） | key name（Chinese） | key value type | key value & other notes | remark
事件命名规则 | 模块名_操作；全小写英文；举例：content_item_impression 信息流_单条内容_曝光
属性命名规则 | 全小写英文 每个完整单词间加下划线；举例：entry_way 进入内容中心方式
属性值规则 | 优先小写英文 每个完整单词间加下划线；举例：minus1_feed 负一屏进入
常用属性名
position | 位置
type | 类型
source | 来源
page | 页面
function | 功能
style | 样式
duration | 时长
status | 状态
常用属性值类型 | 字符串：最常用，属性值含义一目了然
数值型：可进行计算，比如时长类上报可用
bool：仅包含0和1两个值，常表示“是否”
备注填写内容 | 新增事件&属性的版本

```
