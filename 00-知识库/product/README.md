## 数据产品核心能力
1. **价值判断能力**：判断做不做，比怎么做更重要，识别真需求和伪需求
2. **指标体系设计**：从业务目标到可执行指标的拆解能力，OSM+UJM方法论
3. **数据需求转化**：将业务问题转化为数据可解问题，把模糊需求变清晰
4. **产品方案设计**：平衡用户体验、技术可行性、业务价值，给出最优方案
5. **AI协作能力**：用好AI工具提效，设计AI Native产品形态

## 工作全流程
- 问题定义：识别真实用户问题 → 判断价值和优先级
- 方案设计：指标设计 → 产品方案 → 埋点设计 → PRD撰写
- 开发上线：需求评审 → 开发跟进 → 灰度发布 → 数据监控
- 迭代优化：效果评估 → AB实验验证 → 复盘沉淀 → 下一轮迭代
