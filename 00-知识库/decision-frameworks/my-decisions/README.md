# 我的决策案例库

> 作用：沉淀所有重要决策的完整记录（决策简报 + 依据 + 回看结果），按决策类型分类，方便AI检索相似案例、学习判断模式。
> 关联：护栏见 `../methods/龚云荷决策护栏.md`、剧本见 `../methods/龚云荷决策剧本库.md`、输出模板见 `../../../TEMPLATES/决策/决策简报.md`。

## 决策归档规则

1. **每个决策单独存为一个文件**，命名：`YYYY-MM-DD-决策标题.md`
2. **文件内容直接使用决策简报模板**，回看结果追加到文件末尾
3. **`DECISIONS.md` 只保留索引表格**，详情链接到这里的对应文件
4. **新剧本触发条件**：某类决策累计出现3次，提炼为新剧本追加到 `../methods/龚云荷决策剧本库.md`
5. **护栏更新触发条件**：某决策回头看判定为❌推翻/⚠️部分成立，检查是否需要补护栏到 `../methods/龚云荷决策护栏.md`
