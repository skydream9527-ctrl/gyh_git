# 人群包资产登记表

> 每次人群包交付后登记一行，形成可追溯、可复用的资产，避免"交付即失联"。
> 隐私提示：oaid 明细属隐私数据，按规范私发，本表只登记元信息，不存明细。

## 字段说明

| 字段 | 含义 |
|------|------|
| 人群包名称 | 业务可读名称 |
| 交付物 | 万象人群包 ID / HDFS 文件 / 仅量级 |
| 圈选条件摘要 | 核心标签 + 排除 + 阈值 |
| 量级 | 覆盖设备数 |
| 更新频率 | 一次性 / 每日调度 |
| 需求方 | 提需人 |
| 用途 | push / 售卖 / 激励 / 测算 |
| 状态 | 在用 / 已下线 |
