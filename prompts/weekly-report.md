# 周报生成 Prompt 模板

## 使用方式

按模板结构描述本周工作，或者直接说「帮我把这周的工作写成周报」，AI 会从最近的对话记忆中提取素材。

---

## Prompt

> 执行前先读 MEMORY.md 和 memory/ 中本周的记录，已有信息（本周决策、对话摘要）自动补充到周报素材中，不要重复问用户。只问缺失的信息。

我是一名产品经理，请帮我把以下内容整理为周报。

### 本周工作

1. {完成事项 1}
2. {完成事项 2}
3. {进行中事项}

### 关键数据

- {指标 1}：{数值}（{环比变化}）
- {指标 2}：{数值}（{环比变化}）

### 遇到的问题

- {问题 1}：{当前状态 / 需要的帮助}

### 下周计划

1. {计划事项 1}
2. {计划事项 2}

### 输出要求

1. 先总后分：一段总述 + 分段详述
2. 数据用「本周 / 上周 / 变化」三列呈现
3. 问题和风险单独分段，标注是否需要上级介入
4. 语言简洁，每条不超过 2 行
