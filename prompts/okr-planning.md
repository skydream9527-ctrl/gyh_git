# OKR / 季度规划 Prompt 模板

## 使用方式

季度初制定 OKR 或做规划时使用。复制以下 Prompt，替换 `{ }` 中的内容。

---

## Prompt

> 执行前先读 USER.md 和 MEMORY.md，了解我的角色、团队和当前重点。

请帮我制定 {季度，例如 2026 Q3} 的 OKR / 季度规划。

### 上季度回顾

{简述上季度的核心成果和不足，或提供上季度 OKR 复盘文档}

### 本季度背景

- 团队情况：{人数、变动}
- 业务目标：{部门/公司级目标}
- 资源约束：{时间、人力、技术限制}
- 必须做的事：{老板明确要求的、承诺的}

### 我的想法

{初步想法，优先级判断，想聚焦什么}

### 输出要求

1. 给出 2-3 个 Objective，每个 Objective 下 2-4 个 Key Result
2. Key Result 必须可量化，有明确的完成标准
3. 按 P0/P1/P2 标注优先级
4. 明确「不做」清单（显式放弃什么）
5. 给出月度里程碑时间线
6. 识别关键风险和依赖
7. 如果资源不足，建议砍掉什么
