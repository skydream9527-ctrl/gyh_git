# 浏览器 PUSH 数据深度分析报告（2026 年 4 月 · V3）

<callout emoji="📊">
**全局核心结论（V3 升级版）**
1. **人群侧**：50\~80 岁是绝对核心，发送 29.3% → 点击 **57.4%**，转化率 **7.65%**。男性 + 中高收入 + 一线城市效率分别加成 27% / 45% / 25%。
2. **内容侧（新增）**：**「严肃内容」高度匹配核心人群**——国际时政、国际军情、中东局势、战争、银行、经济政策等类目转化率 **1.3%\~1.9%**，是娱乐/生活类大流量类目（0.10%\~0.18%）的 **10 倍以上**。
3. **跨维度共振**：「中老年男性」画像偏好「国际时政/军事/经济」严肃议题，**人群与内容呈强一致性**——这是优化的最大确定性。
4. **资源错配**：娱乐资讯（3036 万曝光、转化 0.16%）、生活记录（2077 万、0.12%）、明星八卦（2583 万、0.69%）等娱乐类目占用大量曝光资源但转化极低，**约 8500 万曝光被浪费**。
5. **画像未知人群**仍是噪声主因，多维度转化率指数 0.13-0.47，建议系统性降权。
</callout>

## 三、年龄维度：50\~80 岁是金牌人群

<callout emoji="🔥">
**结论：50\~80 岁是 PUSH 业务的绝对金牌人群**
- **占比迁移**：发送 29.34% → 曝光 34.52% → 点击 **57.36%**（指数 **1.96**），是唯一在三轮中显著「逆向放大」的群体。
- **转化效率**：曝光→点击转化率 **7.65%**，是整体均值（4.55%）的 1.68 倍，是 18-21 岁年轻人群（2.16%）的 **3.5 倍**。
- 年龄越大，效率越高的规律非常清晰：转化率从 18-21 岁阶梯式上升到 50-80 岁。
- **「未知」年龄人群是最大噪声源**：发送占 14.64%、点击仅 2.42%、转化率 1.36%。
</callout>

## 九、跨维度共振：人群×内容一致性

<callout emoji="🎯">
**核心发现：人群画像与内容偏好高度耦合**  
将 Sheet 3 画像分布与 Sheet 4 内容类目交叉对照，得到清晰的双向印证。「中老年男性 + 中高收入 + 一线城市」用户对「国际时政 / 军事 / 财经 / 法律 / 社会民生」内容有强偏好，**这是 PUSH 业务的「最大公约数」**。
</callout>

| 高效维度 | Sheet 3 人群画像 | Sheet 4 内容类目 |
|-|-|-|
| 年龄偏好 | 50\~80 岁（转化率 7.65%） | 国际时政、军情、战争、经济政策（1.3-1.9%） |
| 性别偏好 | 男性（70.44% 点击占比） | 武器装备、银行、股票证券、中东局势 |
| 收入偏好 | 5000+ 中高收入（6.4%） | 银行、经济政策、股票证券、国际财经 |
| 城市偏好 | 一线/京沪津（5.65%+） | 政治经济财经类强相关 |

<callout emoji="🎯">
**反向佐证：低效内容对应低效人群**
- 娱乐资讯、生活记录、明星八卦、穿衣搭配等低效内容，主要消费人群在 18-29 岁年轻群体（转化率 2.16-2.62%）和女性（4.13%）。
- 这部分人群本身转化效率低，叠加内容吸引力不足，**双重低效**形成资源黑洞。
- 但需注意：年轻人群与娱乐内容是「未来用户基本盘」，不能完全砍掉，应做精细化运营而非粗放降权。
</callout>
