# 指标口径字典

> 内容生态核心业务指标的定义与计算口径。权威参照，减少口头反复澄清。
> 完整口径详解和 SQL 模板见 [project/data-product-workspace/00-知识库/数据资产/指标口径字典/](../project/data-product-workspace/00-知识库/数据资产/指标口径字典/核心指标口径.md)。

## 核心指标速查

| 指标 | 定义 | 计算方式 | 注意事项 |
|---|---|---|---|
| DAU | 日活跃去重设备 | `COUNT(DISTINCT did)` | 区分启动方式（主启/push/三方吊起） |
| 有效 DAU | 信息流有效活跃设备 | 见专用 SQL 模板 | 有效行为含曝光/点击/时长门槛；浏览器与内容中心分开统计 |
| 使用时长 | 设备使用总时长 | `SUM(duration)` | 内容中心限制 0-24 小时；注意单位（秒/分钟） |
| 人均时长 | 人均使用时长 | `SUM(duration) / COUNT(DISTINCT did)` | 分母口径需与 DAU 对齐 |
| UV | 去重访问设备 | `COUNT(DISTINCT did)` 或 oaid | did 与 oaid 粒度不同，人群包用 oaid |
| 次日留存 | Day2 仍活跃占比 | 次日活跃设备 / 当日设备 | 需明确留存基准（新增/全量） |
| 人均搜索次数 | 搜索频次 | `搜索 PV / 搜索 UV` | 注意 Pad 搜索词是否计入 |
| 财收/流水 | 计提收入 | 按算法来源/分 CP 汇总 | 区分算法来源、剔除内投、月计提 vs 季度结算 |
